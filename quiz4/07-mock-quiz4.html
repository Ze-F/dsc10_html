📝 Mock Quiz A · 拼接版（限时 30 分钟）

从 8 份历年 Quiz 4/5 抽取符合 Lec 18-22 范围的题目，组合而成的模拟卷。限时 30 分钟，不查答案。

Problem 1（Lec 18）· Sample Size

你要估计 UCSD 校园树木中橡树（oak）的比例（population parameter）。要做 95% CI 宽度不超过 0.04，最少需要采集多少棵树？

  1. (4 · 0.5 / 0.04)² = 2500
  2. (2 · 0.5 / 0.04)² = 625
  3. (4 · 1 / 0.04)² = 10000
  4. 2 / 0.04 = 50
答案

A. 2500。Polling proportion 用 SD 上界 0.5；n ≥ (4·SD/w)² = (4·0.5/0.04)² = 2500。

注意干扰项 B 错在用 2（应该是 4，因为 CI 是 ±2·SD/√n）；C 错在用 SD=1（不是上界）；D 是开根号没平方。

Problem 2（Lec 18）· Multinomial 一行代码

UCSD 树木 35% eucalyptus、25% pine、40% other。写一行代码，模拟从这个分布按比例抽 40 棵树（with replacement），输出长度 3 的 array，元素分别是 eucalyptus、pine、other 的数量。

答案 
np.random.multinomial(40, [0.35, 0.25, 0.40])

注意：(1) 概率数组要按 eucalyptus、pine、other 顺序；(2) 第二个参数和必为 1；(3) 不要除以 40（要 counts 不要 proportion）。

Problem 3（Lec 19）· Test Statistic 选择 ⭐⭐⭐

测两 grader Jeffrey 和 Kate：

  • Null: 两人 score 同分布
  • Alt: Jeffrey 给的分更低

下面哪些 test statistic 可以从 empirical distribution 直接算 p-value？多选：

  1. Kate's mean − Jeffrey's mean
  2. Jeffrey's mean − Kate's mean
  3. |Kate's mean − Jeffrey's mean|
  4. TVD of two distributions
答案 & 解析

答案：A 和 B。

  • A ✓：Jeffrey 低 → Kate − Jeffrey 大 → 右尾
  • B ✓：Jeffrey 低 → Jeffrey − Kate 负 → 左尾
  • C ✗：abs 后无法区分 Jeffrey 低 vs Kate 低
  • D ✗：scores 是数值，TVD 只用于 categorical

Problem 4（Lec 20）· p-value 决策 ⭐⭐⭐

p-value 评估为 0.07。结论？

  1. Reject at both 0.01 and 0.05
  2. Fail to reject at 0.01, reject at 0.05
  3. Fail to reject at both 0.01 and 0.05
  4. Accept the null at 0.05
答案

答案：C。p = 0.07 > 0.05 → fail to reject at both。D 错（永远不 accept）。

Problem 5（Lec 20-21）· TVD 手算 ⭐⭐

计算 dist1 = [0.20, 0.30, 0.50] 和 dist2 = [0.40, 0.10, 0.50] 之间的 TVD。

答案

|0.20−0.40| + |0.30−0.10| + |0.50−0.50| = 0.20 + 0.20 + 0 = 0.40
TVD = 0.40 / 2 = 0.20

Problem 6（Lec 21）· CI 反推决策 ⭐⭐⭐

声称 "true proportion = 0.50"。95% bootstrap CI 算出来是 [0.52, 0.58]。在 5% 显著性下结论？

答案

0.50 不在 [0.52, 0.58] 内 → reject the null。
声称的 0.50 与 sample 不一致。

Problem 7（Lec 22）· Permutation 代码 ⭐⭐⭐

研究 UCSD 男女生平均 sleep 时间是否不同。DataFrame sleep 有 'gender' 列（'M' 或 'F'）和 'hours' 列。Alt: 男生平均 sleep 时间不同于女生（双边）。Test stat: |mean_M − mean_F|。填空：

def one_diff(df):
    means = df.groupby(___(a)___).mean().get('hours')
    return ___(b)___

observed = one_diff(sleep.assign(shuf=sleep.get(___(c)___)))

diffs = np.array([])
for i in np.arange(10000):
    shuf = ___(d)___

答案

📋 Winter 2026 Quiz 4 完整真题（限时 30 分钟）

Wi26 Q4 的 scope 是 Lec 17-21，最贴近本次 Quiz 4 范围。完整刷一遍，再核对。

📌 设定：grading DataFrame 有 50 行（Jeffrey 25 + Kate 25），列 'grader', 'score', 'submission_id'。Detective Peter 怀疑其中一位评分太严。

Problem 1（Lec 20）

Peter 决定做 permutation test。下面哪个最佳描述 (a) 为什么用 permutation test，(b) 怎么做？

  1. 两 sample 测同分布。Under null，shuffle 组别 labels（如果同分布，labels 不该有意义）
  2. 两 sample 测同分布。Under null，resample with replacement from combined 50 scores
  3. 比较 Jeffrey scores 和已知 Quiz 3 population distribution。Under null，shuffle scores 重算
  4. Shuffling 生成两个 random sample，估计 sampling distribution
答案：A

B 错是因为 resample with replacement 是 bootstrap，不是 permutation。C 错没 known population。D 错把 permutation 当 bootstrap。

Problem 2.1（Lec 20）

Peter 怀疑 Jeffrey 比 Kate 严（Jeffrey 给的分更低）。选 null/alt：

  1. Null: 同分布。Alt: Jeffrey 的 distribution 平均更小。
  2. Null: Jeffrey 的 mean = 70。Alt: ≠ 70。
  3. Null: 同分布。Alt: Jeffrey 的 distribution 完全在 Kate 之下。
  4. Null: Jeffrey 的 mean = Kate 的 mean。Alt: ≠。
答案：A

Problem 2.2（Lec 20）

哪些 test stat 可以直接算 p-value？多选：

  1. Kate's mean − Jeffrey's mean
  2. Jeffrey's mean − Kate's mean
  3. |两组 mean 的差|
  4. TVD of two score distributions
  5. None of the above
答案：A 和 B

C 不对（abs 后单边 alt 无法判方向）；D 不对（scores 数值，TVD 只用于 categorical）。

Problem 3（Lec 21）

填 (a) (b)：

observed_stat = (abs(grading[grading.get('grader') == 'Kate']
                  .get('score').mean())
                 - abs(grading[grading.get('grader') == 'Jeffrey']
                  .get('score').mean()))

p_value = (simulated_stats ___(a)___ observed_stat).sum() / ___(b)___
答案
  • (a) >=
  • (b) 10000（permutation 次数）

解释：observed_stat = abs(Kate's mean) − abs(Jeffrey's mean) = Kate's mean − Jeffrey's mean（score 正数，abs 无效）。Jeffrey 严 → 这个差大 → 右尾。

Problem 4（Lec 18）

估计 Jeffrey 给分低于 70 的比例。95% CLT-based CI。

(a) CI 宽度 ≤ 0.08，最少 sample size?
(b) CI 宽度 ≤ 0.04，最少 sample 是 (a) 的几倍？

答案

(a) (4·0.5/0.08)² = 625

(b) 4 倍（w 减半 → n 变 4 倍）

Problem 5（Lec 19）

Jeffrey 说真比例 = 0.20。95% CI = [0.14, 0.40]。

5.1 5% 显著性下结论？

5.2 为什么？

答案

5.1：Fail to reject（B）。永不 Accept。

5.2：A："Because 0.20 lies inside the CI, the data are consistent with Jeffrey's claim"

Problem 6（Lec 20）

TVD 是什么？

答案：D. 两个 categorical distribution 距离的度量

🔥 25 题陷阱速刷（每题 1 分钟）

所有题目都是 T/F 或单选。考前最后过一遍。

1T/F：Null hypothesis 必须是 well-defined probability model，能 simulate 才行。
答T。不能 simulate 的 hypothesis（如 "the coin is unfair"）只能当 alt。
2T/F：可以"Accept the null"。
答F。永远只能 "fail to reject"。
3T/F：p-value 是 "null 为真的概率"。
答F。p-value 是"在 null 为真前提下，看到 observed 或更极端的概率"。
4Test stat 选 abs vs 不带 abs：alt 是 "A > B" 该用哪种？
答不带 abs（A − B 或 B − A 都可以；要看右尾或左尾）。
5p = 0.04，95% 显著性下结论？
答Reject the null（0.04 < 0.05）。
6p = 0.04，99% 显著性下结论？
答Fail to reject（0.04 > 0.01）。
7T/F：TVD 可以用于比较 smokers vs non-smokers 婴儿的出生重分布。
答F。birth weight 是数值，TVD 只用于 categorical。
8TVD 公式有除以 2 吗？
答有。np.abs(d1 - d2).sum() / 2。
9TVD 的 range？
答[0, 1]。0 = 完全相同，1 = 完全无 overlap。
10用 95% CI 做 hypothesis test，对应的显著性是？
答5% (α = 0.05)。一般地：(100−p)% CI ↔ p% α。
11声称 mean = 5。95% CI = [4.8, 5.2]。5% 显著性下结论？
答Fail to reject（5 ∈ CI）。
12Polling proportion 时 SD 取什么作为上界？
答0.5（√(p·(1-p)) 在 p=0.5 最大）。
13CI 宽度 w 减半，sample size 怎么变？
答变成原来 4 倍（n ∝ 1/w²）。
14T/F：Permutation test 用 df.sample(n, replace=True)。
答F。那是 bootstrap。Permutation 用 np.random.permutation（不放回 shuffle）。
15np.random.multinomial(50, [0.3, 0.7]) 返回什么？
答长度为 2 的 counts array（不是 proportion）。如 array([15, 35])。
16测两 sample 同分布、知道是 numerical 数据，用什么 test stat？
答difference of means（或 abs 版，按双边/单边）。不是 TVD。
17Permutation test 中应该 shuffle group label 还是 value column？
答都可以（概念上）。具体看代码结构。
18T/F：permutation test reject 后能推 causation。
答F。只是相关，不是因果（Deflategate caveat）。
19"95% of intervals contain truth" 是 CI 的正确解读吗？
答True（这是 Quiz 3 重点；Quiz 4 也可能延伸考）。
20Hypothesis test "highly significant" 指 p < ?
答p < 0.01。
21T/F：p-value 算用 > 还是 >= 都对。
答F。必须 >=（包含 observed 本身："as or more extreme"）。
22"68% CI 宽度 ≤ 0.01" 用什么公式算 sample size？
答

68% CI ⇒ ±1·SD/√n（不是 ±2）。所以 width = 2·SD/√n ≤ 0.01 → n ≥ (2·0.5/0.01)² = 10000。
（这是 Wi25 Q4 P1 原题，常见误用 4·SD/√n —— 那是 95% CI 的公式）

23测一枚硬币是否偏 tails (alt 单边)。Test stat 用 |heads − 200| 对吗？
答不对。单边 alt 不该用 abs。要用 number of heads（看左尾）或 number of tails（看右尾）。
24T/F：A/B test 在 DSC 10 等同于 permutation test。
答T（Lec 22 明确说"For us, they mean the same thing"）。
25用 np.random.multinomial 抽样后做 TVD test，count 还是 proportion？
答Proportion。要把 counts 除以 sample size。TVD 输入必须是 proportion（和为 1）。
