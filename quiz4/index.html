综合 📝 真题 ⏱️ 30min/套

🔑 必背 8 件套（贴在桌前）

  1. Hypothesis test 四步：(1) 写下 null / alt；(2) 选 test statistic；(3) under null 模拟，得到 empirical distribution；(4) 看 observed 落在哪里 → 算 p-value → 决策
  2. Null hypothesis 必须是 well-defined probability model（可以从中 simulate）；alt 可以是任何"不一样"的说法
  3. 选 test statistic：alt 是 "A > B" 或 "A < B" → 不带 abs；alt 是 "different / not equal" → 带 abs
  4. p-value = under null，看到 observed 或更极端的概率。p < 0.05 → reject；p < 0.01 → highly significant。永不 "accept" null，只能 "fail to reject"
  5. TVD = np.abs(dist1 - dist2).sum() / 2，只用于 categorical distribution（多 category）
  6. CI 反推 test：要测 "param = x"（双边），构造 (100−p)% CI；x 在 CI 内 → fail to reject；不在 → reject
  7. Permutation test：用 np.random.permutation(col) 打乱 group labels；不需要知道 population；test stat 通常是 difference / abs difference of means
  8. 三个 np.random 别混：choice（从 array 抽样）、multinomial（按概率抽 categorical counts）、permutation（重新排列）

⚠️ Top 5 高频陷阱（贴脑门上）

  1. Test stat 方向搞反：Wi26 Q4 选项陷阱。alt = "Jeffrey 比 Kate 严" → 用 "Kate − Jeffrey"（这样 Jeffrey 严的话 stat 越大），不能反方向
  2. TVD 不能用于 numerical 变量：smoking & birth weight 是数值的 → 不能用 TVD，要用 difference of means
  3. Permutation 不是 bootstrap：permutation 是 shuffle labels（不放回），bootstrap 是 resample with replacement
  4. "Accept" the null 永远是错的：只能 "fail to reject" 或 "reject"
  5. CI 测的是 "param = x" 双边：单边 alt（">" 或 "<"）不能直接用 CI 法
