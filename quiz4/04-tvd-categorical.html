Section 4 · TVD for Categorical Distributions

覆盖讲义：Lec 20-21（Total Variation Distance） ｜ Quiz 4 出现率：⭐⭐ Tier 2（Fa24、Wi26、Sp24 Q5 都直接考） ｜ 预计学习时间：1 小时

🎯 本节定位：TVD 是"用 categorical distribution 做 hypothesis test"时的 test statistic。本节核心要做到三件事：(1) 会按手算 TVD（很简单的公式）；(2) 知道它只用于 categorical distribution（Wi26 Q4 P6 直接考）；(3) 会按代码模板跑一个 TVD test（Sp24 Q5 P2 直接考）。

📋 考点速览

考点Tier一句话
TVD 公式 ⭐⭐ np.abs(dist1 - dist2).sum() / 2
TVD 只用于 categorical distribution ⭐⭐⭐ Wi26 Q4 P6 直接考；Lec 21 summary 强调"NOT used for permutation tests"
"除以 2" 的原因 ⭐⭐ over-representation 总量 = under-representation 总量；两者相等，所以除以 2 表示"距离"
TVD ∈ [0, 1] ⭐ 0 = 两个分布完全一样；1 = 完全无 overlap

📚 知识点

1. TVD 是什么

Total Variation Distance (TVD)
= sum of absolute differences of proportions, divided by 2

对两个 categorical distribution（同样 k 个类别，按相同顺序排）：

TVD(dist1, dist2) = Σ|dist1[i] − dist2[i]| / 2

用 numpy 一行写：

def total_variation_distance(dist1, dist2):
    return np.abs(dist1 - dist2).sum() / 2

2. 手算示例（Lec 20 Alameda jury）

EthnicityEligible (dist1)Panels (dist2)Difference|Difference|
Asian0.150.26+0.110.11
Black0.180.08−0.100.10
Latino0.120.08−0.040.04
White0.540.540.000.00
Other0.010.04+0.030.03
Sum0（永远 = 0）0.28

TVD = 0.28 / 2 = 0.14。

3. 为什么除以 2

注意三个事实：

  1. 两个 distribution 都是 proportion，各自和 = 1。
  2. 所以差值 (dist2 − dist1) 的和 = 0。
  3. 正差值之和 = 负差值之和的绝对值（互补）。

具体到 Alameda 例子：

  • 正差值（Panels 比 Eligible 多的类别）：Asian +0.11、Other +0.03 → 总 over-representation = 0.14
  • 负差值（Panels 比 Eligible 少的类别）：Black −0.10、Latino −0.04 → 总 under-representation = 0.14

正负两边数值必然相等。所以 sum(|diff|) 会把这个 0.14 算两次。除以 2 才是真正的"距离"。

4. TVD 的几何含义

三种等价说法（任选一种背）：
  • TVD = total amount of over-representation across all categories
  • TVD = total amount of under-representation across all categories
  • TVD = "half of the sum of absolute differences" between two categorical distributions

5. ⭐ TVD 只用于 categorical distribution

这是 Wi26 Q4 P6 的精确考点。考试给的选项里：
  • ❌ "TVD 是两个 group 的均值差" —— 错（那是 difference in means）
  • ❌ "TVD 是两个分布的距离"（不说明类型）—— 错，太泛
  • ❌ "TVD 是两个 numerical distribution 的距离" —— 错
  • ✅ "TVD 是两个 categorical distribution 的距离" —— 对

为什么不能用于 numerical？因为 numerical 是连续的，没有"类别比例"概念。TVD 公式 Σ|p1 − p2| 在 numerical 上没意义。

这也是 Lec 22 强调的："TVD is not used for permutation tests" —— permutation test 通常处理两个数值 sample（如 smokers' baby weights vs non-smokers'），不能用 TVD。permutation test 该用 difference of means / abs diff of means。

6. TVD 作为 test statistic 的特性

  • TVD ≥ 0（绝对值的和）—— 永远非负
  • TVD ≤ 1（两个分布完全无 overlap 时为 1）
  • TVD = 0 当且仅当两个分布完全相同
  • 所以 p-value 看右尾：TVD 越大 → 越不像 null（"分布相同"）→ alt 越可能

📌 注意：TVD 自带 abs 性质，不需要再加 abs。直接看大值偏 alt。

7. 用 TVD 做 hypothesis test 的代码模板

套路：null = "panels 是从 eligible distribution 随机抽的"，test stat = TVD between sample distribution and eligible distribution。

# Setup
eligible = np.array([0.15, 0.18, 0.12, 0.54, 0.01])  # 5 categories
observed_panel = np.array([0.26, 0.08, 0.08, 0.54, 0.04])
n = 1453  # sample size

def total_variation_distance(dist1, dist2):
    return np.abs(dist1 - dist2).sum() / 2

# Step 1: observed TVD
observed_tvd = total_variation_distance(observed_panel, eligible)  # 0.14

# Step 2: simulate under null
tvds = np.array([])
for i in np.arange(10000):
    sample_counts = np.random.multinomial(n, eligible)
    sample_dist = sample_counts / n                              # 转 proportion
    new_tvd = total_variation_distance(sample_dist, eligible)
    tvds = np.append(tvds, new_tvd)

# Step 3: p-value（TVD 看右尾）
p_value = np.count_nonzero(tvds >= observed_tvd) / 10000
# Alameda 案例里 ≈ 0.0 → reject null

8. 一图速记（背在脑子里）

Categorical hypothesis test 三步：
  1. np.random.multinomial(n, null_dist) / n → 一次模拟的 sample distribution
  2. total_variation_distance(sample, null_dist) → 一次 TVD
  3. np.count_nonzero(tvds >= observed_tvd) / N → p-value

⚠️ 常见陷阱与考点

陷阱 1：⭐ 忘了除以 2

很多人写成 np.abs(d1 - d2).sum() —— 这不是 TVD，是"sum of absolute differences"。考试代码题里这两个会作为对比选项出现。

陷阱 2：⭐ TVD 用于 numerical 数据

Smoking & birth weight、bike prices、Patriots pressure drops 这些都是 numerical → 不能用 TVD。要用 difference of (sample) means / medians。

陷阱 3：忘了把 counts 转 proportion

np.random.multinomial(1453, eligible) 返回 counts（如 [220, 260, ...]），不是 proportion！要算 TVD 必须先 / 1453。
Sp24 Q5 P2.1 的代码框就在考这里 —— 必须填 np.random.multinomial(500, proposed) / 500。

陷阱 4：两个 distribution 类别顺序不一致

TVD 假设两个 array 按相同顺序排对应类别。如果一个是 [Asian, Black, ...]，另一个是 [Black, Asian, ...]，结果就错了。

陷阱 5：把 TVD 当 difference of means

Wi26 Q4 P6 干扰选项之一 "TVD 是 group mean 的差"。错。Group mean 的差是 difference of means，用于 permutation test；TVD 是 categorical 分布的距离。

陷阱 6：用 TVD 时还加 abs

TVD 内部已经 abs 了。Test stat 写成 abs(observed_tvd) 等于多此一举（TVD 本来就 ≥ 0）。不算错但多余。

陷阱 7：TVD 的 p-value 用左尾

错。TVD 大 = 分布不同（偏 alt） → 永远是右尾 >=。考试选项里会有干扰的左尾版本，立刻排除。

陷阱 8：把 sample size 和 distribution 长度搞混

np.random.multinomial(500, [0.15, 0.20, 0.40, 0.20, 0.05]) —— 500 是 sample size（题目里 bikes 有 500 行），不是 5（类别数）。

真题 1：Fa24 Q4 P3 ⭐⭐（手算 TVD）

Fall 2024 Quiz 4, Problem 3 (Lec 21)

Jack 和 Eric 各自看过的电影分布如下：

MusicalComedyActionHorrorTotal
Jack2142220
Eric55151515100

求 Jack 和 Eric 的"genre 分布"之间的 TVD。

显示答案 & 解析

先把 counts 转 proportions（必须！）：

MusicalComedyActionHorror
Jack2/20 = 0.1014/20 = 0.702/20 = 0.102/20 = 0.10
Eric55/100 = 0.5515/100 = 0.1515/100 = 0.1515/100 = 0.15
|diff|0.450.550.050.05

Σ|diff| = 0.45 + 0.55 + 0.05 + 0.05 = 1.10

TVD = 1.10 / 2 = 0.55

陷阱：忘了 / 2 会答 1.10；忘了转 proportion 会答 51 / 2（用 counts 算）—— 都错。

真题 2：Wi26 Q4 P6 ⭐⭐⭐（TVD 定义选项陷阱）

Winter 2026 Quiz 4, Problem 6 (Lec 20)

下面哪个是 TVD 的正确描述？

  1. The difference between two group means.
  2. A measure of how different two distributions (of any type) are.
  3. A measure of how different two numeric distributions are.
  4. A measure of how different two categorical distributions are.
  5. The proportion of simulated statistics that are at least as extreme as observed.
  6. The width of a 95% confidence interval.
显示答案

答案：D。

  • A 错：是 difference of means。
  • B 错：缺少"categorical"限制，太泛。
  • C 错：方向反了，TVD 不能用于 numeric。
  • D 对。
  • E 错：这是 p-value 的定义。
  • F 错：这是 CI width。

真题 3：Sp24 Q5 P2 ⭐⭐（TVD 完整 hypothesis test）

Spring 2024 Quiz 5, Problem 2 (Lec 21)

Bikes 500 行，5 种 style。Proposed: [0.15, 0.20, 0.40, 0.20, 0.05]。Observed: [0.20, 0.20, 0.30, 0.20, 0.10]。用 TVD 测：

def total_variation_distance(distr1, distr2):
    return __(a)__

proposed = np.array([0.15, 0.20, 0.40, 0.20, 0.05])
observed = np.array([0.20, 0.20, 0.30, 0.20, 0.10])
tvds = np.array([])
for i in np.arange(10000):
    simulated = np.random.multinomial(__(b)__, __(c)__) / 500
    new_tvd = total_variation_distance(__(d)__, __(e)__)
    tvds = np.append(tvds, new_tvd)

observed_tvd = total_variation_distance(__(f)__, __(g)__)

# 哪个是正确的 p-value？
# A. np.count_nonzero(tvds >= observed_tvd) / len(observed)
# B. np.count_nonzero(tvds >= observed_tvd) / len(tvds)
# C. np.count_nonzero(tvds <= observed_tvd) / len(observed)
# D. np.count_nonzero(tvds <= observed_tvd) / len(tvds)
显示答案
  • (a) np.abs(distr1 - distr2).sum() / 2
  • (b) 500（sample size）
  • (c) proposed（null 假设下的分布）
  • (d), (e) simulated, proposed（或反过来；TVD 对称）
  • (f), (g) observed, proposed（同样可以反过来）
  • p-value: B（tvds >= observed_tvd 右尾、分母用 len(tvds)）

陷阱：(b) 容易写成 5（类别数）—— 错；要写 500 (sample size)。p-value 选项里 C/D 用 <= 是干扰，TVD 永远右尾。A/C 分母用 len(observed)（=5），也是干扰。

练习 4：判断该用 TVD 还是 difference of means

综合判断

下列每个场景，写出适合的 test statistic（TVD / abs diff of means / diff of means / 其他）：

  1. Alameda County 5 种 ethnicity 的 panel 是否随机
  2. Smokers vs non-smokers' baby weights 是否同分布
  3. Patriots vs Colts 的 pressure drop（数值）是否一样
  4. Movies 中 Comedy/Action/Horror/Musical 4 类是否符合预期比例
  5. Bikes 价格在 road vs hybrid 间是否有差（permutation）
显示答案
  1. TVD（categorical 比较）
  2. Difference of means（numeric, two-sample）
  3. Difference of means（numeric）
  4. TVD（categorical）
  5. Difference of means（permutation 涉及 numeric data）

📌 一句话规则：categorical 比较 → TVD；numeric 两 sample 比较 → diff (or abs diff) of means / medians。

练习 5：手算 TVD

综合练习

计算下列两个分布的 TVD：

Dist1 (DSC 10): [0.45 Freshman, 0.35 Sophomore, 0.15 Junior, 0.05 Senior+]

Dist2 (DSC 40A): [0.15 Freshman, 0.35 Sophomore, 0.35 Junior, 0.15 Senior+]

显示答案

|diff|: |0.45−0.15| + |0.35−0.35| + |0.15−0.35| + |0.05−0.15| = 0.30 + 0 + 0.20 + 0.10 = 0.60

TVD = 0.60 / 2 = 0.30

（这道题是 Lec 20 / 21 的 concept check）

练习 6：TVD = 0 / TVD = 1 的极端情况

综合理解
  1. 什么情况下 TVD = 0？
  2. 什么情况下 TVD = 1？给一个例子。
  3. TVD 能 > 1 吗？
显示答案
  1. 两个分布完全相同（所有类别比例相同）
  2. 两个分布无 overlap。例如 dist1 = [1, 0, 0]，dist2 = [0, 0.5, 0.5]。
    |diff| = 1 + 0.5 + 0.5 = 2，TVD = 2/2 = 1
  3. 不能。两个 proportion vector 都满足和为 1，差值的绝对值和不可能 > 2，所以 TVD ≤ 1
