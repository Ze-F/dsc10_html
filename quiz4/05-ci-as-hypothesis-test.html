Section 5 · CI as Hypothesis Test

覆盖讲义：Lec 21（CI for Hypothesis Testing） ｜ Quiz 4 出现率：⭐⭐⭐ Tier 1（Wi26 Q4 P5 直接考；Wi25 Q4 P3.5 T/F） ｜ 预计学习时间：45 分钟 - 1 小时

🎯 本节定位：这是 Quiz 4 的"小巧但高频"知识点。Lec 21 教了一个巧妙的捷径：当 null 形如 "param = some value"（且 alt 是双边）时，我们不用 simulate，直接用 Quiz 3 学过的 CI 就能做 hypothesis test。Wi26 Q4 P5 直接考这个方法。
本节短，但必须把"什么时候能用 CI"和"如何匹配 cutoff 与 confidence level"搞清楚。

📋 考点速览

考点Tier一句话
适用条件：null = "param = x"（双边） ⭐⭐⭐ 仅当 null 是某个 single value（如 "mean = 98.6"），alt 是 "param ≠ x" 时可用
(100−p)% CI 配 p% 显著性 ⭐⭐⭐ p = 0.05 → 95% CI；p = 0.01 → 99% CI
决策：x 在 CI 内 → fail to reject ⭐⭐⭐ x ∈ CI 说明 x 是 plausible value；x ∉ CI → reject
构造 CI 的两条路径 ⭐⭐ Bootstrap CI（任何 statistic）｜ CLT-based CI（仅 mean）
什么时候不能用 CI 法 ⭐⭐⭐ (1) Alt 是单边 (">" 或 "<")；(2) Null 涉及两个 sample 比较（应用 permutation）；(3) Null 不是 "param = x" 形式

📚 知识点

1. 核心 idea：CI 给的就是 "plausible values"

Quiz 3 学过：c% CI 给出 population parameter 的 c% plausible range。

Hypothesis testing 在问："null 提出的 param = x 这个值是否 plausible？"

把这两个一拼：

构造一个 (100 − p)% CI ↔ 等价于 p% significance level 的 hypothesis test
  • 如果 x ∈ CI → x 是 plausible → 与 null 一致 → fail to reject null
  • 如果 x ∉ CI → x 不 plausible → 与 null 不一致 → reject null

2. 严格的适用条件

必须同时满足三条：
  1. Null 形式是 "population parameter = some value x"（如 "mean = 98.6"、"proportion = 0.5"）
  2. Alt 形式是 "population parameter ≠ x"（双边，不是 ">" 或 "<"）
  3. 能构造该 parameter 的 CI（用 bootstrap 或 CLT）

⚠️ 这个方法不能用于：

  • Permutation test 类问题（two-sample comparison，如 smoking & birth weight）
  • Categorical distribution 比较（TVD test）
  • 单边 alt 题（如 "score 比 70 高的比例 > 0.20"）—— 严格来说不能直接用

3. 匹配 cutoff 和 confidence level

显著性 (α)对应 CIBootstrap percentilesCLT 公式
5% (α = 0.05) 95% CI 2.5, 97.5 mean ± 2·SD/√n
1% (α = 0.01) 99% CI 0.5, 99.5 mean ± 2.576·SD/√n
10% (α = 0.10) 90% CI 5, 95 mean ± 1.645·SD/√n

📌 记法：α + c = 100%。
在 DSC 10 范围内最常用 95% CI ↔ 0.05 显著性。

4. 完整例子：Body Temperature 98.6°F（Lec 21）

  • 大家都说人类平均体温是 98.6°F
  • 研究者收集了 130 个人的样本，sample mean ≈ 98.25，sample SD ≈ 0.73
  • 问：sample 是否与 "population mean = 98.6" 这个 null 一致？

4.1 用 Bootstrap CI

resample_means = np.array([])
for i in np.arange(10000):
    resample = temperatures.sample(130, replace=True)
    resample_means = np.append(resample_means,
                                resample.get('temperature').mean())

# 95% CI
[np.percentile(resample_means, 2.5),
 np.percentile(resample_means, 97.5)]
# ≈ [98.126, 98.375]

4.2 用 CLT CI

sample_mean = 98.25  # 实际是 98.249...
sample_sd = 0.73
n = 130

SE = sample_sd / np.sqrt(n)  # ≈ 0.064
[sample_mean - 2*SE, sample_mean + 2*SE]
# ≈ [98.121, 98.377]

4.3 决策

两种方法 CI 几乎一致：约 [98.12, 98.38]。98.6 不在这个 CI 内 → reject the null at the 5% significance level。

结论：sample 不支持 "人类平均体温 = 98.6°F" 的说法（事实上现代 Yale 的研究也指出原始 98.6 是 Wunderlich 老旧温度计的产物，真实约 97.9）。

5. Wi26 Q4 P5 的完整逻辑（最高频考法）

题目说：Jeffrey 反驳"分数 < 70 的真比例 = 0.20"。研究者构造 95% CI 得到 [0.14, 0.40]。问 5% 显著性下结论。

  • Null：proportion = 0.20
  • Alt：proportion ≠ 0.20（双边）
  • α = 0.05 → 95% CI（匹配！）
  • 0.20 ∈ [0.14, 0.40] → x 是 plausible value → fail to reject

正确解释："because 0.20 lies inside the confidence interval, the data are consistent with Jeffrey's claim"。

⚠️ Wi26 Q4 P5.1 干扰选项里有 "Accept" —— 错。永远只能 fail to reject。

6. 何时不能用 CI 法（Wi25 Q4 P3.5）

Wi25 Q4 的整道大题里，null 是 "每个 field 80% 概率被 fertilized"，alt 是 "不是 80%"。P3.5 问 T/F: "We could construct a CI to test these hypotheses"。

表面看：null 是 "param = 0.80"，看起来能用 CI 法。但实际更细想：

  • 这里 sample 已经在 hand，sample proportion 可以算
  • 但 sample proportion 本身就是 estimate；对它做 bootstrap CI / CLT CI 来看 0.80 是否在内 —— 完全 OK
  • 所以答案是 True（可以）

关键判断 —— 是否能用 CI 法：

能 ✅：null = "param = x"，且 sample 直接对应该 param。例如：
- 体温 mean = 98.6（sample 是体温数据，可 bootstrap mean）
- Proportion = 0.20（sample 是 0/1 数据，可 bootstrap proportion）

不能 ❌：
- Two-sample comparison（smoking vs non-smoking）→ permutation
- Categorical distribution match → TVD
- Single-sided alt（"> x"）严格来说也不能直接用 CI（虽然有变通办法）

⚠️ 常见陷阱与考点

陷阱 1：⭐ "Accept the null" 选项

Wi26 Q4 P5.1 出现 4 选项：Reject / Fail to reject / Accept / Not enough info。"Accept" 永远错。

陷阱 2：⭐ Cutoff 和 confidence level 不匹配

"95% CI ↔ 0.05 显著"。考题如果说 "test at 1% significance"，应该用 99% CI，不是 95%。Wi26 Q4 P5 都是用 95% CI ↔ 5% α，比较 standard。

陷阱 3：⭐ Null 不是 "param = x" 形式还用 CI

permutation 类问题（"smokers' babies 比 non-smokers' 轻"）不能用 CI 法 —— 因为没有"single value"做 null。要用 Lec 22 的 permutation test。

陷阱 4：方向混淆

"x 在 CI 内 → fail to reject"，不是 reject。这是反方向最常见的低级失误。

陷阱 5：误用 CI 的解释

Wi26 Q4 P5.2 干扰选项 "Because the midpoint of the interval is 0.27, we have evidence against Jeffrey's claim"。错 —— midpoint 在 reject 判断里不重要，重要的是 x 是否落在 CI 内。

陷阱 6：把 "CI 宽" 当 "no power"

Wi26 Q4 P5.2 另一个干扰选项 "the interval has width 0.26 which is much wider than we wanted ... cannot reject any claim"。错。CI 宽度跟"能否 reject"无关；只看 x 是否在 CI 内。

陷阱 7：单边 alt 想用 CI

严格意义上 CI 法是双边判断（"param = x" 双边 alt）。如果 alt 是 "param > x"，标准做法是 simulate（不是直接用 CI）。
当然 DSC 10 范围内只要看到 "param = x" 类 null 就可以用 CI；如果 alt 是单边，题目通常会指明用 simulation。

陷阱 8：忘了构造哪个 CI

CI 是对 population parameter 的估计区间，不是对 sample statistic 的。Sample mean 是确定的（计算后是单值），不需要 CI。

真题 1：Wi26 Q4 P5 ⭐⭐⭐（CI 反推决策）

Winter 2026 Quiz 4, Problem 5 (Lec 21)

Jeffrey 主张"分数 < 70 的真比例 = 0.20"。Detective Peter 构造 95% CI 得到 [0.14, 0.40]。

5.1：在 5% 显著性下，多选下面正确的：

  1. Reject
  2. Fail to reject
  3. Accept
  4. Not enough information

5.2：哪个最准确解释了这个结论？

  1. Because 0.20 lies inside the CI, the data are consistent with Jeffrey's claim.
  2. Because 0.20 lies inside the CI, the data are inconsistent with Jeffrey's claim.
  3. Because the interval has width 0.26 which is wider than we wanted, we don't have enough precision to reject any claim.
  4. Because the midpoint of the interval is 0.27, we have evidence against Jeffrey's claim.
显示答案 & 解析

5.1 答案：B（Fail to reject）

  • 0.20 在 [0.14, 0.40] 内 → fail to reject null
  • C "Accept" 永远错（hypothesis test 从不 accept）
  • D 不对，信息够

5.2 答案：A

  • A 对：0.20 在 CI 内 → consistent
  • B 错：方向反了
  • C 错：CI 宽度跟决策无关
  • D 错：midpoint 跟决策无关

Winter 2025 Quiz 4, Problem 3.5 (Lec 21)

给定 null/alt：null = "每个 flower field 80% 概率被 fertilized"，alt = "不是"。能否构造 CI 来 test？

T / F?

显示答案

答案：True。

因为 null 形式是 "proportion = 0.80"（single value），alt 是 "≠ 0.80"（双边）→ 满足 CI 法的两个条件。

练习 3：判断能否用 CI 法

综合判断

下面每个 hypothesis test，判断能否用 CI 法（而不是 simulate）：

  1. Null: 人类平均身高 = 170 cm。Alt: ≠ 170 cm
  2. Null: smoker's babies 和 non-smoker's babies 平均出生重相同。Alt: smoker's 更轻
  3. Null: Alameda jury panel 是从 eligible 分布随机抽的。Alt: 不是
  4. Null: 这枚硬币 P(heads) = 0.5。Alt: P(heads) ≠ 0.5
  5. Null: 美国总统得票率 = 50%。Alt: > 50%
显示答案
  1. 能。Null 是 "mean = 170"，双边 alt → 95% CI 看 170 是否在内
  2. 不能。这是 two-sample 比较 + 单边 alt → permutation test
  3. 不能。比较 categorical distributions（5 类）→ TVD test
  4. 能。Null 是 "P = 0.5"，双边 → bootstrap / CLT CI for proportion
  5. 不能（严格意义）。Alt 是单边 ("> 50%")，CI 法默认双边。但实际操作里如果 50% 落在 CI 一端外，依然可以下结论

练习 4：综合应用

综合 - Section 1 复用

研究 UCSD 学生平均 sleep 时间。Sample n=100，sample mean = 6.8 小时，sample SD = 1.5 小时。Null: 平均 sleep = 7 小时。在 5% 显著性下测：

显示答案

用 CLT-based CI（Lec 17）：

  • SE = 1.5 / √100 = 0.15
  • 95% CI = [6.8 − 2·0.15, 6.8 + 2·0.15] = [6.5, 7.1]
  • 7 在 [6.5, 7.1] 内 → fail to reject null at 5% level

结论：数据与"平均 sleep = 7 小时"一致。

练习 5：从 CI 反推 sample 信息

Section 1 + Section 5 综合

已知 95% CLT CI 为 [10.4, 13.6]，sample SD = 4 days。求 sample mean 和 sample size。

显示答案

（这是 Sp25 Q4 P3 原题）

  • CI midpoint = sample mean = (10.4 + 13.6) / 2 = 12
  • CI half-width = 2 · SD/√n = 13.6 − 12 = 1.6
  • 解出：2·4/√n = 1.6 → √n = 5 → n = 25
