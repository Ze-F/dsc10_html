📋 考点速览

本节是每份midterm必考的Tier 1核心。函数 + .apply 通常以"填空题"形式出现，常和 .assign、.groupby、字符串方法（.split）、条件分支（if/elif/else）一起组合考察。

  • ⭐⭐⭐ 自定义函数语法：def / 参数 / return。
  • ⭐⭐⭐ Series.apply(func)：把函数作用到Series每个元素，返回新Series。
  • ⭐⭐⭐ 标准模式 df = df.assign(NewCol = df.get('OldCol').apply(my_func))。
  • ⭐⭐⭐ 条件分支函数（if/elif/else），常配 .apply 做分类。
  • ⭐⭐ 字符串解析：.split(' ')[0]、.strip("'")。
  • ⭐⭐ .reset_index()：把index变回普通列，才能对它 .apply。
  • ⭐⭐ Local scope（局部作用域）—— 函数内定义的变量出函数后访问会 NameError。
  • ⭐ print 与 return 的区别。

📚 知识点

1. 函数定义语法

def function_name(param1, param2):
    '''docstring：简短描述函数功能（可选但推荐）'''
    # 函数体
    result = param1 + param2
    return result          # 用 return 把值返回出来
  • 参数 (parameters)：param1, param2 是占位符（仅在函数内部可见）。
  • 实参 (arguments)：调用时传入的真实值，如 function_name(3, 4)。
  • docstring：用三引号包起来的说明文字。
  • return：必须有，否则函数返回 None。

2. return vs print ⚠️

def pythagorean(a, b):
    c = (a**2 + b**2) ** 0.5
    print(c)              # 只显示，不返回
    # 没有 return → 隐式返回 None

def better_pythagorean(a, b):
    c = (a**2 + b**2) ** 0.5
    return c              # 把值送出去，可以赋值给变量

x = pythagorean(3, 4)            # 屏幕显示 5.0；x = None
y = better_pythagorean(3, 4)     # y = 5.0，可以再用 y + 10

考点：如果函数只有 print 而没有 return，赋值后变量是 None，再做运算会 TypeError。

3. Local Scope（局部作用域）

def triple(x):
    return x * 3

triple(5)   # 15，函数内 x 暂时 = 5
x           # NameError: name 'x' is not defined
            # 函数外没有 x！

函数参数和函数内部变量都是局部的——函数执行完就消失，外部访问会报 NameError。

4. 0 个 / 1 个 / 多个参数

def greeting():                       # 0 个参数
    return 'Hi!'

def triple(x):                        # 1 个参数
    return x * 3

def custom_multiples(k, how_many):    # 2 个参数
    return np.arange(k, (how_many + 1) * k, k)

custom_multiples(10, 7)   # array([10, 20, 30, 40, 50, 60, 70])

5. Series.apply(func) —— 核心 ⭐⭐⭐

对Series里的每个元素调用一次 func，返回一个新的Series。

# 标准用法：传函数名（不要加括号！）
df.get('col').apply(my_func)

# 错误示例：
df.get('col').apply(my_func())   # ❌ 这是先调用 my_func()，把它的返回值传给 apply
df.get('col').apply(my_func)     # ✅ 把函数本身传进去
⚠️ 重要规则（babypandas）：
  • .apply 只能用在 Series 上，不能用在 DataFrame 上。
  • 必须先用 df.get('col') 拿到Series，再 .apply。
  • 传函数名，不带括号。
  • 输出还是Series（长度和原Series一样）。

6. 完整工作流：define → apply → assign

# 步骤1：先在单个例子上把逻辑写对
'Pradeep Khosla'.split(' ')[0]    # 'Pradeep'

# 步骤2：把逻辑封装成函数
def first_name(full_name):
    '''从完整姓名中取出 first name'''
    return full_name.split(' ')[0]

# 步骤3：用 .apply 作用到整列，并用 .assign 加新列
roster = roster.assign(
    first = roster.get('name').apply(first_name)
)
# 现在 roster 多了一列 'first'

7. 在function中使用 if / elif / else

def categorize(price):
    if price <= 6:
        return 'Cheap'
    elif price <= 14:
        return 'Moderate'
    else:
        return 'Expensive'

dining.assign(
    Affordability = dining.get('Price').apply(categorize)
)

这是 midterm 最常考 的填空模式：3 段式分类函数 + .apply + .assign。

8. .reset_index() —— 给index"复活"成列

name_counts.index           # Index([...])，不是 Series
name_counts.index.apply(max)   # ❌ AttributeError: 'Index' has no attribute 'apply'

# 解决方案：把 index 变成普通列
name_counts.reset_index().get('first').apply(max)   # ✅ OK

当你想对 index 里的字符串做 .apply 时，必须先 .reset_index()。

9. 字符串解析常见操作

'820 Opal Street, San Diego, CA, 92109'.split(', ')
# ['820 Opal Street', 'San Diego', 'CA', '92109']

'820 Opal Street'.split(' ')[0]   # '820'
'820 Opal Street'.split(' ', 1)   # 注意：split 第二参数是最大次数

"'getting'".strip("'")            # 'getting'，去掉两端的引号
'13:15'.split(':')                # ['13', '15'] → 拿到时和分

⚠️ 常见陷阱与考点

陷阱 1：传函数名 vs 调用函数
.apply(my_func) ✅ 把函数本身传给 apply
.apply(my_func()) ❌ 先调用 my_func（缺参数会报错），把返回值传过去
记忆口诀：apply 后面跟函数名，不要加括号。
陷阱 2：.apply 是 Series 方法，不是 DataFrame 方法
在 babypandas 里，df.apply(...) 不存在！必须先 df.get('col') 拿到Series。
df.get('col').apply(func) ✅
df.apply(func) ❌
陷阱 3：函数没写 return → 返回 None
def f(x):
    y = x * 2
    print(y)        # 没有 return！

result = f(5)       # 屏幕显示 10
result + 1          # TypeError: unsupported operand type(s) for +: 'NoneType' and 'int'
很多 midterm 题目陷阱是要求"返回"，但学生写成"打印"。
陷阱 4：Local scope —— 函数内变量出不来
def f(x):
    z = x * 2
    return z

f(5)
z       # NameError！z 只在函数内部存在
考点：选择题问"以下哪个会报错"，常拿 local scope 出题。
陷阱 5：要对 index 做 apply，必须先 .reset_index()
groupby 后的结果常常以分组列作为 index。要在那一列上 apply 函数，先把它变回普通列： 
grouped = df.groupby('city').count()
grouped.reset_index().get('city').apply(my_func)   # ✅
陷阱 6：.apply 返回新Series，原数据不变
df.get('col').apply(f) 不会修改原 DataFrame。要"加列"必须用 .assign： 
df = df.assign(new = df.get('col').apply(f))   # 必须重新赋值给 df
陷阱 7：.split 返回 list，可以直接索引
'a-b-c'.split('-') → ['a', 'b', 'c']
'a-b-c'.split('-')[1] → 'b'
若要转 int：int('a-b-c'.split('-')[1])。
💡 万能模板（背下来）： 
# 模板A：定义函数 → apply → assign 加新列
def my_func(x):
    # ...处理 x...
    return result

df = df.assign(NewCol = df.get('OldCol').apply(my_func))

# 模板B：条件分类函数
def classify(x):
    if condition1:
        return 'A'
    elif condition2:
        return 'B'
    else:
        return 'C'

df = df.assign(Category = df.get('col').apply(classify))
