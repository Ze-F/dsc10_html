Problem 1（Lec 4-5）· DataFrame Indexing + Querying ⭐⭐⭐

P1 DataFrame 综合 ~8 分钟

DataFrame cars（mock）有列：'model' (str index), 'year' (int), 'price' (float), 'mpg' (float), 'class' (str, 如 "Compact"/"Midsize")。已设 'model' 为 index。

(a) 哪个表达式返回 "Prius" 这一行的 mpg？

  1. cars.loc['Prius'].get('mpg')
  2. cars.iloc['Prius'].get('mpg')
  3. cars.get('mpg').loc['Prius']
  4. cars['Prius']['mpg']

(b) 写一行：返回所有 2010 年后（含 2010）且 mpg > 40 的Compact 车的 DataFrame。

(c) 写一行：在所有 Midsize 车中，按 price 降序找最贵那辆的 model 名（一个 string）。

答案 & 解析

(a) A 和 C 都对。

  • A：.loc['Prius'] 返回 Series → .get('mpg') 取 mpg
  • B ✗：.iloc 用整数，不能传 string
  • C：先 .get('mpg') 拿 mpg Series（仍以 model 为 index）→ .loc['Prius'] 取该 model
  • D ✗：babypandas 没有方括号取 column 的语法

(b)： 

cars[(cars.get('year') >= 2010) & (cars.get('mpg') > 40) & (cars.get('class') == 'Compact')]
⚠️ 三个条件用 & 连，每个用括号包好（precedence 陷阱）。不能用 and。

(c)： 

cars[cars.get('class') == 'Midsize'].sort_values(by='price', ascending=False).index[0]
关键：(1) 先过滤 Midsize；(2) sort_values 降序；(3) 因为 'model' 已经是 index，第一行的 index 值 = 最贵那辆的 model。用 .index[0]（不是 .iloc[0].get('model')，因为 model 不在列里）。

Problem 2（Lec 7）· Density Histogram ⭐⭐⭐

P2 Histogram bin height ~7 分钟

500 名 UCSD 学生，drawn 一个 density histogram of 'hours_studied'，bins = np.arange(0, 41, 5)。bin [10, 15) 的 height = 0.06。

(a) 多少学生 study 时间在 [10, 15)？

(b) 如果改 bins = np.arange(0, 41, 10)，bin [10, 20) 包含原 [10, 15) 和 [15, 20)（后者 height = 0.04）。新 bin 的 height？

答案 & 解析

(a) bin width = 5，proportion = height × width = 0.06 × 5 = 0.30。count = 0.30 × 500 = 150 名学生。

(b) 新 bin 总 proportion = 0.30 + 0.04·5 = 0.30 + 0.20 = 0.50。新 width = 10。
新 height = 0.50 / 10 = 0.05。

⚠️ 高频陷阱：rebin 后高度不是简单平均。要先算总 proportion 再除以新 width。
density histogram 公式：Height = Proportion / Width，所有 bin 总面积 = 1。

Problem 3（Lec 8-9）· Apply + GroupBy + Merge ⭐⭐⭐

P3 apply + groupby + merge 综合 ~10 分钟

DataFrame orders 有列 'customer_id'、'product'、'amount'。DataFrame customers 以 'customer_id' 为 index，有列 'name' 和 'tier' (其中 tier 是 "gold"/"silver"/"bronze")。

(a) 写一个函数 price_range(amount) 返回 string："low"（≤50）、"mid"（50-200）、"high"（>200）。然后用它给 orders 加一列 'range'。

(b) 用 groupby(['tier', 'range']) 统计每个 (tier, range) 组合下的订单数。哪个语法对？

  1. merged = orders.merge(customers, left_on='customer_id', right_index=True); merged.groupby(['tier', 'range']).count()
  2. orders.groupby(['tier', 'range']).count()
  3. customers.groupby(['tier', 'range']).count()

(c) 若 orders 有 1000 行、customers 有 200 行（每个 customer 在 orders 中至少出现 1 次），orders.merge(customers, ...) 后预测行数？

答案 & 解析

(a)： 

def price_range(amount):
    if amount <= 50:
        return "low"
    elif amount <= 200:
        return "mid"
    else:
        return "high"

orders = orders.assign(range=orders.get('amount').apply(price_range))
⚠️ .apply(price_range) 不带括号（传函数名不是调用）。

(b) A。orders 没有 tier 列，必须先 merge 进 customers 的信息。
B ✗（缺 tier 列）、C ✗（customers 没 range 列）。

(c) 1000 行。每个 order 与 unique customer 1对1 匹配（inner join），orders 全部保留。
⚠️ 不是 1000 × 200 = 200000，不是 1200（不是 union）。merge 的行数 = 匹配次数。

Problem 4（Lec 11-12）· Probability + Simulation ⭐⭐⭐

P4 概率 + 模拟 ~10 分钟

一个袋子里 5 红、3 蓝、2 绿球。

(a) 无放回抽 3 个，全是红球的概率？

(b) 有放回抽 3 个，至少 1 个蓝球的概率？

(c) 写代码：用 simulation 估算 (b) 的概率（10000 次重复，输出一个数）。

答案 & 解析

(a)：5/10 × 4/9 × 3/8 = 60/720 = 1/12 ≈ 0.0833。

(b)：P(at least 1 blue) = 1 − P(no blue) = 1 − (7/10)³ = 1 − 0.343 = 0.657。

(c)： 

colors = np.array(['R']*5 + ['B']*3 + ['G']*2)
counter = 0
for i in np.arange(10000):
    draws = np.random.choice(colors, 3, replace=True)
    if np.count_nonzero(draws == 'B') >= 1:
        counter = counter + 1
counter / 10000
⚠️ 关键 4 个点：(1) replace=True；(2) at least 1 用 >= 1；(3) np.count_nonzero(draws == 'B') 数 True；(4) 最后除以 10000。

Problem 5（Lec 14-15）· Bootstrap CI + 解读 ⭐⭐⭐

P5 Bootstrap + CI 解读 ~10 分钟

有 my_sample（n=200）。要用 bootstrap 构造 92% CI for population median of 'income'。

(a) 填空（10000 次 bootstrap）：

boot_medians = np.array([])
for i in np.arange(10000):
    resample = my_sample.sample(___(i)___, replace=___(ii)___)
    boot_medians = np.append(boot_medians, resample.get('income').___(iii)___)

ci = [np.percentile(boot_medians, ___(iv)___),
      np.percentile(boot_medians, ___(v)___)]

(b) 算出的 92% CI 是 [42000, 48000]。下面哪些表述正确？多选：

  1. 92% probability that the true median income is in [42000, 48000].
  2. 92% of all households have income between 42000 and 48000.
  3. If we repeated this entire sampling-and-bootstrap process many times, about 92% of resulting intervals would contain the true median income.
  4. The sample median is between 42000 and 48000.
答案 & 解析

(a)：(i) my_sample.shape[0]，(ii) True，(iii) median()，(iv) 4，(v) 96。
92% CI → 砍掉两端各 4% → percentile (100−92)/2 = 4 和 (100+92)/2 = 96。

(b) 答案：C 和 D。

  • A ❌ "probability" 陷阱——CI 和 truth 都固定，不是概率语义。
  • B ❌ "households" 陷阱——CI 是 median 分布的中间 92%，不是个体值的范围。
  • C ✅ 唯一正确的"process"解读。
  • D ✅ sample median 在 bootstrap 分布中心附近，大概率落 CI 内（除非极端情况）。

Problem 6（Lec 16-17）· Chebyshev / Normal / CLT ⭐⭐⭐

P6 Chebyshev / CLT 综合 ~10 分钟

UCSD 学生 GPA 的 population mean = 3.2，SD = 0.6（任意分布）。

(a) 用 Chebyshev：至多多少比例的学生 GPA 在 [2.0, 4.4] 范围之外？

(b) 若 GPA 服从 normal 分布，多少比例的 GPA 在 [2.0, 4.4] 之内？

(c) 抽取 n=144 学生的 sample。sample mean GPA 的 SD（即 SE）是多少？sample mean 在 [3.1, 3.3] 内的概率？（GPA 任意分布，但 n=144 够大用 CLT）

答案 & 解析

(a)：[2.0, 4.4] 是 mean ± 2 SD（2.0 = 3.2 − 2·0.6，4.4 = 3.2 + 2·0.6）。
Chebyshev：z=2 → 至少 1 − 1/4 = 75% 在范围内 → 至多 25% 在范围之外。

(b)：Normal 68/95/99.7 法则：±2 SD → ~95% 之内。

(c)：SE = pop SD / √n = 0.6 / 12 = 0.05。
sample mean 分布近似 normal（CLT），mean = 3.2，SD = 0.05。
[3.1, 3.3] = 3.2 ± 0.1 = 3.2 ± 2·SE = 2 个 SE 范围 → 约 95%。

⚠️ SD vs SE 关键区分：(a)(b) 用 population SD = 0.6；(c) 用 SE = 0.05（sample mean 的 SD）。混了就全错。

Problem 7（Lec 18-20）· Sample Size + Hypothesis Testing ⭐⭐⭐

P7 Sample size + p-value ~10 分钟

(a) 你要估 UCSD 学生中 vegetarian 的比例，目标：95% CI 宽度 ≤ 0.06。最少要采集多少学生？

(b) 你做了一个 hypothesis test：

  • Null：UCSD 学生中 vegetarian 比例 = 0.25
  • Alt：UCSD 学生中 vegetarian 比例 ≠ 0.25
  • Test statistic：|sample proportion − 0.25|

得到 p-value = 0.03。下面哪些结论正确？多选：

  1. Reject null at 0.05 significance.
  2. Reject null at 0.01 significance.
  3. Fail to reject null at 0.05 significance.
  4. Accept the null at 0.05 significance.
  5. If null is true, the probability of seeing data as or more extreme is about 3%.
答案 & 解析

(a)：proportion polling，用 0/1 数据 SD 上界 = 0.5。
n ≥ (4·SD/w)² = (4·0.5/0.06)² = (2/0.06)² = 33.33² = 1112 学生（向上取整）。

(b) 答案：A 和 E。

  • A ✅ p = 0.03 < 0.05 → reject
  • B ❌ p = 0.03 > 0.01 → fail to reject at 0.01
  • C ❌（在 0.05 下要 reject）
  • D ❌ 永远不 accept——只能 fail to reject
  • E ✅ 这是 p-value 的正确解读

Problem 8（Lec 21-22）· CI as Hypothesis Test + Permutation ⭐⭐⭐

P8 CI 反推 + 选工具 ~10 分钟

(a) 你声称"UCSD 学生 vegetarian 比例 = 0.20"。用 bootstrap 算出 95% CI = [0.22, 0.28]。在 5% 显著性下决策？

(b) 下面 4 个问题，每个最合适的工具是？（选 hypothesis testing / permutation testing / CLT-CI / regression）

  • (i) UCSD 男生平均身高是不是 175 cm？
  • (ii) UCSD 男生和女生平均身高有差异吗？
  • (iii) UCSD 学生 GPA 跟睡眠时长有关系吗？
  • (iv) UCSD 学生中 vegetarian 比例的范围是多少？
答案 & 解析

(a)：0.20 不在 [0.22, 0.28] 内 → reject null。声称的 0.20 与数据不一致。

(b)：

  • (i) hypothesis testing（含 CLT-CI 反推 also OK）—— null: mean = 175，单 sample vs 已知 value
  • (ii) permutation testing —— 两组比较，没有 known population，shuffle group labels
  • (iii) regression —— 两个 continuous 变量的关系
  • (iv) CLT-CI 或 bootstrap CI —— 估 population proportion

⚠️ 工具选择决策树：单样本 vs 已知值 → hypothesis；两样本是否同分布 → permutation；估 parameter → CI；两连续变量关系 → regression。

Problem 9（Lec 23-24）· Correlation + Regression ⭐⭐⭐

P9 r + 单位换算 + 反推 ~12 分钟

UCSD 课程的 'midterm' 和 'final' 成绩 linearly associated，r = 0.7。midterm：mean = 75，SD = 10。final：mean = 70，SD = 15。

(a) 求 regression line 在 original units 下的 slope m 和 intercept b（predict final from midterm）。

(b) 某学生 midterm 90 分。预测他 final 多少分？

(c) 若把 midterm 和 final 成绩都按 1.2 倍重新 scaling（grade inflation），下面三个量怎么变？

  • r：
  • slope（predicting new final from new midterm）：
  • intercept：

(d) 真实情况：某学生 midterm 90、final 80。residual？

答案 & 解析

(a)：m = r · SD_y / SD_x = 0.7 · 15 / 10 = 1.05。
b = mean_y − m · mean_x = 70 − 1.05 · 75 = 70 − 78.75 = −8.75。

(b)：predicted final = 1.05 · 90 + (−8.75) = 94.5 − 8.75 = 85.75。

(c)：x 和 y 同时 × 1.2：

  • r: stay same（线性变换不变）
  • slope: stay same（SD_y 和 SD_x 同步 × 1.2，比例不变）
  • intercept: × 1.2（b = mean_y − m·mean_x，mean_y 和 mean_x 都 × 1.2，slope 不变 → b × 1.2）→ −10.5

(d)：residual = actual − predicted = 80 − 85.75 = −5.75。
⚠️ 不要写 +5.75（"predicted − actual" 是错的顺序）。

Problem 10（Lec 25）· Residual Plot 解读 + Prediction Interval ⭐⭐⭐

P10 Residuals 综合 ~8 分钟

用 weight 预测 mpg of cars。数据 n = 200，mean weight = 3000 lbs。residual plot 显示明显的 U 形（先正、后负、再正）。

(a) 从这个 residual plot 可以得出哪些结论？多选：

  1. r 一定接近 0。
  2. 线性 fit 不是 best model；可能 nonlinear 更好。
  3. regression line 的 RMSE 比某条其他直线大。
  4. residual plot 是 patternless 的。

(b) 你 bootstrap 算了一组 prediction intervals。weight = 2500、3000、3500 中，哪个对应的 prediction interval 最窄？

答案 & 解析

(a) 答案：B 只有。

  • A ❌ curved residual 不暗示 r 弱
  • B ✅ 唯一可靠结论
  • C ❌ regression line 就是 RMSE 最小的直线
  • D ❌ 显然有 U 形 pattern

(b)：weight = 3000（等于 mean_x，最窄）。3500 离 mean 500，2500 离 mean 500，两者一样宽；3000 最窄。

Midterm 范围（Lec 1-12）· 8 题

T1babypandas 取列用什么语法？Lec 4

答案

df.get('col')。不能用 df['col']。.get(['col']) 返回单列 DataFrame，.get('col') 返回 Series。

T2.loc[5] vs .iloc[5]？Lec 4

答案

.loc 按 index label（可以是 string 或 int label）；.iloc 按 整数位置。如果 index 是 0/1/2/3...，两者结果一样；如果 index 被 set 成 string，.iloc 仍按位置。

T3布尔过滤用 and 还是 &？Lec 5

答案

&（和 | 配对）。每个条件用括号包：df[(df.get('a')>0) & (df.get('b')<5)]。不能用 and/or。

T4density histogram 的 height 是什么？Lec 7

答案

Height = Proportion / Width。所有 bin 面积之和 = 1。bin 是 [a, b) 半开区间（最后一个右端闭合）。

T5series.apply(func) vs series.apply(func())？Lec 8

答案

必须传函数名（不带括号）：series.apply(my_func)。带括号是先调用 my_func() 再把返回值传给 apply，错。

T6merge 默认是 inner join 还是 outer join？没匹配的行怎么样？Lec 9

答案

默认 inner join。没匹配的行消失。预测行数 = 两边 join key 配对总次数（不是简单的 max/sum）。

T7np.append(arr, x) 要不要重新赋值？Lec 10

答案

必须重新赋值：arr = np.append(arr, x)。np.append 返回新 array，不改原 array。

T8"至少 1 个" 概率怎么算？Lec 11

答案

用 complement：P(at least 1) = 1 − P(none)。如 "至少 1 个红"（n 次有放回，红比例 p）= 1 − (1−p)n。

Quiz 3 范围（Lec 13-18）· 8 题

T9SRS 必须 with replacement 还是 without？Lec 13

答案

SRS = Simple Random Sample = without replacement，且每行有相同概率被选。df.sample(n) 默认 without replacement = SRS。df.sample(n, replace=True) 不是 SRS。

T10Bootstrap loop 三件套？Lec 14

答案

(1) resample = sample.sample(sample.shape[0], replace=True)（同 sample size、with replacement）；(2) 算 statistic 加进 array；(3) 重复 N 次（通常 10000）。

T1192% CI 用哪两个 percentile？Lec 14

答案

(100−c)/2 和 (100+c)/2。所以 92% → percentile 4 和 96（不是 8 和 92）。

T12"X% chance the true mean is in this CI"——这种描述对吗？Lec 15

答案

错。CI 是固定的（已经算出来），truth 也是固定的，没有概率语义。唯一正确描述："X% of intervals generated by this process contain the truth."

T13Chebyshev：z = 2, 3, 4 对应保证多少比例？Lec 16

答案

z=2 → 至少 75%；z=3 → 至少 88.8%（= 8/9）；z=4 → 至少 93.75%（= 15/16）。任何分布都成立。Normal 68/95/99.7 只对 normal。

T1495% CLT-based CI 公式？Lec 17

答案

[mean − 2·SD/√n, mean + 2·SD/√n]，其中 SD 是 sample SD（≈ population SD），n 是 sample size。SE = SD/√n。

T15Sample size for proportion polling（95% CI 宽度 w）？Lec 18

答案

n ≥ (4·SD/w)²。0/1 数据 SD 上界 = 0.5。所以 polling 公式：n ≥ (4·0.5/w)² = (2/w)²。w 是总宽度不是 margin。

T16np.random.multinomial(n, [p1, p2, p3]) 返回什么？Lec 18

答案

长度 3 的 array，元素是 counts（不是 proportions）。p 数组之和必须 = 1。要算 proportion 需自己 / n。

Quiz 4 范围（Lec 18-22）· 8 题

T17Null vs Alt 哪个必须 well-defined probability model？Lec 19

答案

Null 必须是可以从中模拟的明确分布（例如 "fair coin"、"proportion = 0.5"）。Alt 可以是任意"不同"的说法。

T18Alt 是单边 ("A > B") 还是双边 ("A ≠ B") 时 test statistic 带不带 abs？Lec 19

答案

双边 → 带 abs（|A − B|）；单边 → 不带 abs（A − B）。Wi26 Q4 P2.2 直接考。

T19p-value 的精确定义？Lec 20

答案

Under null is true，观察到 "observed 或更极端" 的 test statistic 的概率。不是 P(null is true)。不是 P(under alt)。

T20p-value 决策："Accept the null" 对不对？Lec 20

答案

永远错。只能 reject 或 fail to reject。"accept" 在 hypothesis testing 中是禁词。

T21TVD 公式？只用于哪种数据？Lec 20-21

答案

TVD = np.abs(dist1 − dist2).sum() / 2。只用于 categorical distribution（多个 category 的 proportions）。数值数据用 difference of means。

T22用 CI 反推 hypothesis test 的前提？Lec 21

答案

Alt 必须是 双边（"param ≠ x"）。在 p% 显著性下用 (100−p)% CI。x 在 CI 内 → fail to reject；不在 → reject。单边 alt 不能直接用 CI。

T23Permutation 是 with replacement 还是 without？Lec 22

答案

Permutation = shuffle labels（不放回），每个 row 出现 1 次但 group label 被随机分配。np.random.permutation(col)。Bootstrap = with replacement。两者不能混。

T24p = 0.5 时 "exactly half of simulated stats were less extreme than observed" 对不对？Lec 20

答案

对（Wi25 Q5 P1.2 官方答案）。p-value = count(stats ≥ observed) / N，p = 0.5 → 一半 ≥、另一半 < observed → "exactly half less extreme" 字面成立。

Final 增量范围（Lec 23-25）· 6 题

T25r 的公式（一句话）？Lec 23

答案

r = mean of (x_su · y_su)，即 "x 和 y 都标准化后乘积的均值"。(standard_units(df.get(x)) * standard_units(df.get(y))).mean()。

T26r = 0 等价于 "no association" 吗？Lec 23

答案

不等价。r = 0 只意味着 "no linear association"。Parabola y = x² 有强 association 但 r ≈ 0。

T27Regression line 在 original units 下的 slope 和 intercept 公式？Lec 24

答案

m = r · SD_y / SD_x（SD_y 在分子！），b = mean_y − m · mean_x。Regression line 必过 (mean_x, mean_y)。

T28"95% sample means within 2 SDs of pop mean，due to CLT" Yes 还是 No？Lec 17 / Wi26 Q1.2

答案

Yes, due to CLT（官方答案，措辞陷阱！）。官方解读"2 SDs"为"2 SDs of sample-mean distribution"（即 2 SE）。不是直觉的"2 pop SD"。

T29residual = ? Lec 25

答案

residual = actual y − predicted y。方向不能反！predicted = 5.4、actual = 2.1 → residual = −3.3（不是 +3.3）。

T30Bootstrap prediction interval 在哪里最窄？Lec 25

答案

在 x = mean_x 处最窄；远离时变宽。Sp24 Q12 P2.3：mean = 140，100 vs 120 哪个 wider？答 100（离 mean 更远）。
