📚 知识点

从 inference 到 prediction

课程后半段的主线一直是 statistical inference（用 sample 推 population）：

  • 已学（Lec 13-22，详见 quiz3 + quiz4 模块）：
    • Hypothesis testing（一 sample vs 已知 distribution）
    • Permutation testing（两 sample 是否同 distribution）
    • Bootstrap CI / CLT CI（估 population parameter）
    • 用 CI 反推 hypothesis test 结论
  • Lec 23 起切到 prediction：给一个 sample，能不能对"sample 外"的数据点做预测？
    • 具体方法：linear regression（最简单、最常用的预测工具，data science toolkit 的核心）
    • 路径上要先讲 correlation（两 numerical 变量之间的关联强度）

Final 视角：inference 部分仍然会大量考（quiz3/quiz4 内容是 Final 的主体），prediction（Lec 23-25）是新增内容。每套 Final 通常有 1-2 个 regression 大题（含 correlation 性质、slope/intercept 推导、单位转换、prediction、outlier 影响等），权重不小。

Association：scatter plot 与 positive / negative

预测之前必须先看 scatter plot：

  • Association（关联）：两 numerical 变量之间任意类型的关系；可以是 linear，也可以是 non-linear。
  • Positive association：x 增加，y 也倾向增加（plot 朝右上）。例：'price' vs 'acceleration'（acceleration 这里是 km/h/s，越大表示车越快）。
  • Negative association：x 增加，y 倾向减少（plot 朝右下）。例：'price' vs 'mpg'（油耗好的车通常便宜）。
  • No association：scatter 无 pattern，"a patternless cloud"。

课程里反复强调：分析两个 numerical 变量的第一步永远是画 scatter plot。看图能立刻判断"有没有关系、是 linear 还是 curved、有没有 outlier"。

hybrid.plot(kind='scatter', x='acceleration', y='price');
# y vs. x 的措辞习惯：x 是横轴，y 是纵轴

Correlation coefficient r：定义 + 公式 + 代码

Correlation coefficient r 衡量两变量之间 linear association 的强度。

r  =  mean of ( x(su) · y(su) )

即"把 x 和 y 都转成 standard units 后，对应位置乘积的均值"。

standard units 复习

xi (su)  =  ( xi − mean of x ) / SD of x

标准化后：mean = 0，SD = 1，没有单位。

计算 r 的代码（必背模板）

def standard_units(col):
    return (col - col.mean()) / np.std(col)

def calculate_r(df, x, y):
    """Returns the average value of the product of x and y,
       when both are measured in standard units."""
    x_su = standard_units(df.get(x))
    y_su = standard_units(df.get(y))
    return (x_su * y_su).mean()

⚠️ 注意 np.std vs col.std()：DSC10 一律用 np.std（population SD，分母为 n）。.std() 默认是 sample SD（分母为 n − 1），结果略不同。考试代码用 np.std。

r 的关键性质

这些性质是 Final 的高频考点，全部背熟：

性质说明影响
1. r ∈ [−1, 1] r = 1：完全正向 linear；r = −1：完全负向 linear；r = 0：无 linear pattern。 任何题目里 r 超出 [−1, 1] 都是错的。
2. r 是对称的：r(x, y) = r(y, x) 因为 x_su × y_su = y_su × x_su，乘法可交换。 "r between A and B" 跟"r between B and A"完全一样。但regression line 不对称（predicting y from x 和 predicting x from y 是两条不同的线，下面会讲）。
3. r 对线性变换不变 对 x 做 x → a·x + b（线性变换：单位换算、缩放、平移），r 不变。同样对 y 做线性变换，r 也不变。 price in dollars vs mpg 的 r  =  price in yen vs kpg 的 r。这是高频陷阱题（Wi24 Q3 直接考"换单位后哪些量会变"）。
4. r 没有单位 因为 standard units 把单位除掉了。 r 是纯数字，不带 "dollars"、"inches" 之类。

为什么 r 用 standard units？

关键直觉：线性变换不改变 scatter plot 的形状（只是改变坐标轴的刻度）。既然形状没变，"linear pattern 的强度"就不该变。standard units 是一种统一所有单位的方式：无论 x、y 原本用什么单位，到 standard units 之后都"长得一样"。

r 是对称的：可视化记忆

# 这两个调用结果完全一样
calculate_r(hybrid, 'acceleration', 'price')   # 0.6956
calculate_r(hybrid, 'price', 'acceleration')   # 0.6956

r 只衡量 LINEAR association（最高频陷阱）

r = 0 不等于 "no association"。它只表示没有 linear pattern。两个变量可以有极强的非线性 association，r 仍可能 ≈ 0。

经典反例：抛物线（parabola）

令 x 从 −6 取到 6，y = x²。这是一个完美的非线性关系：

x2 = bpd.DataFrame().assign(
    x=np.arange(-6, 6.1, 0.5),
    y=np.arange(-6, 6.1, 0.5) ** 2
)
x2.plot(kind='scatter', x='x', y='y')
calculate_r(x2, 'x', 'y')   # ≈ 0  (因为对称：左半边 x_su 负 / y_su 正，右半边 x_su 正 / y_su 正，乘积在 -→+ 之间平均掉)
  • 左半 x < 0：x_su 负，y_su 正（因为 y² 大）→ 乘积负
  • 右半 x > 0：x_su 正，y_su 正 → 乘积正
  • 由于左右对称，平均下来 ≈ 0

所以这个图明显有 association（看到 x 几乎能完美预测 y），但 r ≈ 0。

记忆口诀：r 是 linear 的指标；association 是任意类型的关系。
Association ✓ 而 Correlation ✗ 的情况 真实存在（parabola、sin 波、圆形分布等）。

Correlation ≠ Causation

即使 |r| 接近 1，也不表示一个变量"导致"了另一个：

  • 可能是 反向 causation：B → A 而非 A → B。
  • 可能是 第三变量（confounder）同时影响 A 和 B。
  • 可能只是巧合。

r 只回答"用 line 预测的准不准"，不回答"为什么"。

Regression line in standard units（Lec 23 引入）

有了 r，下一步是用一条直线来做预测。当 x、y 都在 standard units 时，所谓 regression line 就是：

predicted y(su)  =  r · x(su)
  • 过原点 (0, 0)：因为 r · 0 = 0。在 standard units 里这意味着 "x = mean of x" → 预测 "y = mean of y"。
  • 斜率 = r，且 r ∈ [−1, 1]，所以这条线斜率绝对值 ≤ 1。
  • 跟"过原点 + 斜率为 1"的 y = x 线对比：regression line 永远更平（除非 |r| = 1）。

⚠️ 不对称：predicting y from x 用的是 predicted y_su = r · x_su；而 predicting x from y 用的是 predicted x_su = r · y_su。这是两条不同的线（即使 r 相同），通常斜率不同（在 original units 里）。

Regression to the mean

核心观察：在 standard units 下，因为 |r| ≤ 1，所以
给一个 x_su，预测的 y_su = r · x_su 比 x_su 更接近 0（即更接近 mean）。
这就是 regression to the mean（回归 = 向均值靠拢）。

含义

  • 妈妈身高 +2 SD（很高），r ≈ 0.32，预测儿子身高 +0.64 SD（仍比平均高，但没那么高）。
  • 妈妈身高 −1 SD（偏矮），预测儿子身高 −0.32 SD（仍偏矮，但没那么矮）。
  • 这是预测，不是事实：实际个体可能远高于或远低于预测。"regression to the mean" 只描述 regression line 这个预测规则的系统性偏向。

为什么称为 "regression"

Galton（19 世纪）发现高个父母的孩子平均没他们那么高、矮父母的孩子也没他们那么矮——后代"回归"群体均值。这种 line-based 预测就以此命名，沿用至今。

⚠️ 常见陷阱与考点

陷阱 1：r = 0 ≠ 无关联

这是 Final 最常考的概念陷阱。看到"两个变量没有 linear association"不等于"两个变量没关系"。parabola、圆形分布、对称的 sin 波等场景 r 都 ≈ 0 但 association 强烈。

考题措辞举例（注意细分）：
  • "There is no association between A and B." —— 强论断，r = 0 不足以推出。
  • "There is no linear association between A and B." —— r = 0 等价。
  • "A and B are uncorrelated." —— r = 0 等价。

陷阱 2：单位换算时哪些量变 / 不变

这是 Wi24 Final Q3 的原题考法。给一份数据，把某列从 kg 换成 lb（线性变换 ×2.2）。下列哪些会变？

量变 / 不变为什么
mean of weight变（× 2.2）mean 是线性的：mean(a·x) = a·mean(x)
SD of weight变（× 2.2）SD 也是线性的（绝对值 / 跨度的概念）
Variance of weight变（× 2.2²）variance = SD²
r (height, weight)不变 ✓r 对线性变换不变
Proportion within 3 SD不变只是单位换了，相对位置不变
Slope of regression line in original units变（× 2.2 或 ÷ 2.2，取决于谁被换）m = r · SDy / SDx；当 y 换单位时 SDy × 2.2 → slope × 2.2
Slope in standard units不变（= r）SU 形式下 slope ≡ r，与原单位无关
Intercept of regression line变b = mean_y − m·mean_x，mean 和 slope 都变了

陷阱 3：r 和 regression line 的对称性

r 对称、regression line 不对称。

  • r(x, y) = r(y, x)。✓
  • 但"predicting y from x"和"predicting x from y"是两条不同的回归线，斜率通常不同（在 original units 里）。✗ 把 y = m·x + b 倒过来变成 x = (1/m)·y − b/m 是错的——倒过来后那条线不是"predicting x from y"的 regression line。
  • 例外：在 standard units 下，predicting y_su from x_su 的斜率是 r；predicting x_su from y_su 的斜率也是 r（因为乘积公式对称）。但两条线"看起来"不同——它们都过 (0,0)、都斜率 r，但变量角色互换了，画在同一坐标系里其实是同一条线反射到 y=x 上。

陷阱 4：r 不告诉你 slope 的具体值

"r = 0.5" 不代表"x 增加 1 单位 y 增加 0.5 单位"。在 standard units 下 slope = r 没错，但在 original units 下 slope 是 m = r · SD_y / SD_x，跟 SD 的比例有关。这点 Section 2 会详讲。

陷阱 5：r 接近 ±1 ≠ 数据真的呈直线

r 衡量"有多接近"一条直线，但 r 接近 1 也只是"接近 perfect linear"。真要确认是否真的 linear，要看 residual plot（Lec 25 内容）。Curved data 配合一两个 outlier，可能伪装出高 r。

陷阱 6：outlier 对 r 的剧烈影响

单个 outlier 可以把 r 从 −0.03 拉到 0.985（Lec 24 PDF 里的真实例子）。所以"画 scatter plot 看 outlier"是任何 regression 分析前的第一步。具体见 Section 2。

📝 往年真题

题目来源：practice.dsc10.com/lectures/lec23/ 收录的真题（按知识点标签聚合），加上近年 Final 的 correlation 相关题。每题答案默认折叠。

真题 1：Wi24 Final Q3 ⭐⭐⭐（单位换算 · 多选）

Winter 2024 Final · Problem 3（多选 · 单位换算后什么会增加）

In olympians, "Weight" is measured in kilograms. There are 2.2 pounds in 1 kilogram. If we converted "Weight" to pounds instead, which of the following quantities would increase? Select all that apply.

  1. The mean of the "Weight" distribution.
  2. The standard deviation of the "Weight" distribution.
  3. The proportion of "Weight" values within 3 standard deviations.
  4. The correlation between "Height" and "Weight".
  5. The slope of the regression line predicting "Weight" from "Height".
  6. The slope of the regression line predicting "Height" from "Weight".
显示答案 & 解析

答案：A、B、E（官方：Options 1, 2, 5）。

  • (A) ✅ mean × 2.2 → 增加（mean 是正的，乘 > 1 的数会增加）
  • (B) ✅ SD × 2.2 → 增加（SD 是非负的，同上）
  • (C) ❌ 不变。换单位是线性变换，标准化后 z-score 不变 → 落在 ±3 SD 内的同样的数据点仍在 ±3 SD 内。
  • (D) ❌ 不变。r 对 x、y 任一线性变换都不变。
  • (E) ✅ slope = r · SD_y / SD_x。这里 y = Weight 被换单位 → SD_y × 2.2，r 不变，SD_x 不变 → slope × 2.2 → 增加。
  • (F) ❌ 减少。这里 x = Weight（被换单位）→ SD_x × 2.2，y = Height（不变）→ slope × (1/2.2) → 减少。

⚠️ 记忆要点：r 不变 是 final 高频陷阱；slope 在 哪个变量被换单位和它在分子还是分母上决定增减。

真题 2：Fa23 Final Q14 P1.2 ⭐⭐（r 值 → scatter plot 配对）

Fall 2023 Final · Problem 14.1.2（scatter plot 识别 · 给定 r = −1/4）

Ashley 有 400 行 txn。"amount" 的 mean 和 SD 都是 70 dollars。同时 "lifetime" 的 mean 和 SD 都是 c dollars。Correlation between transaction amounts 与 lifetime spending = −1/4。下面哪个 scatter plot（A/B/C/D）符合？

（图省略：A 显著正向 / B 轻微负向 / C 显著负向 / D 无 pattern）

显示答案 & 解析

答案：Option B（轻微负向）

  • r = −1/4 → 负的 → 排除显著正向（A）和无 pattern（D）
  • |r| = 0.25 较小 → "弱"的负向关系 → 选项 B 而非 C（C 太强了）
  • 关键判断：r 的符号决定方向，r 的绝对值大小决定"贴线程度"。

真题 3：Wi24 Final Q8 P4.1 ⭐⭐⭐（从预测反推 r · 计算）

Winter 2024 Final · Problem 8.4.1（已知 mean/SD 与一组预测 → 求 r）

In Olympic hockey, # of goals scored is linearly associated with # of attempted shots. 给定：goals 的 mean = 10、SD = 5；shots 的 mean = 30、SD = 10。Regression line predicts 20 shots → 6 goals. 求 r。给精确分数或小数。

显示答案 & 解析

答案：r = 4/5（= 0.8）

用 standard units 形式 y_su = r · x_su：

  • x = 20 shots → x_su = (20 − 30) / 10 = −1
  • predicted y = 6 goals → predicted y_su = (6 − 10) / 5 = −0.8
  • 代入：−0.8 = r · (−1) → r = 0.8 = 4/5

套路：题目里给了"mean、SD、一对 (x, predicted y)"——立刻把它们都转 SU，用 y_su = r · x_su 反解 r。这是 Final 最常见的"反推 r"题型。

真题 4：Wi24 Final Q8 P4.2 ⭐⭐⭐（最大化预测 → r 取什么值）

Winter 2024 Final · Problem 8.4.2（最大化 predicted y 的两小题）

In Olympic baseball: runs 的 mean = 8、SD = 4；hits 的 mean = 24、SD = 6。Regression line predicts runs from hits.

(a) 最大可能的"predicted number of runs for a team that gets 27 hits"？

(b) 让"predicted runs for a team with 25 hits"尽可能大，r 应取什么值？

显示答案 & 解析

(a) 答案：10

predicted runs_su = r · hits_su。hits = 27 → hits_su = (27 − 24) / 6 = 0.5。要最大化 predicted runs_su = r · 0.5，r ∈ [−1, 1]，最大化时 r = 1。

那么 predicted runs_su = 0.5 → predicted runs = 0.5 · SDruns + meanruns = 0.5 · 4 + 8 = 10。

(b) 答案：r = 1

同理，hits = 25 → hits_su = (25 − 24) / 6 = 1/6 > 0。要让 predicted runs_su = r · (1/6) 最大，取 r = 1。

陷阱：如果题目里 x_su 是负的（比如 hits = 22 → hits_su = −1/3），那么要让 predicted y 最大，r 应取 −1（让乘积变成正的最大值）。
口诀：要预测最大 → 让 r 与 x_su 同号且都取 ±1。

真题 5：Wi24 Final Q8 ⭐（T/F · slope in SU 的范围）

Winter 2024 Final · Problem 8 (T/F prelude)（standard units 下 slope 的范围）

True or False：The slope of the regression line, when both variables are measured in standard units, is never more than 1.

显示答案 & 解析

答案：True

在 SU 下 slope ≡ r ∈ [−1, 1]，所以 slope ≤ 1 永远成立（严格说："never more than 1" 是 ≤ 1，对 r = 1 也对）。

同理：slope in SU ≥ −1 也永远成立。

真题 6：Wi26 Final Q11 (a) ⭐⭐⭐（regression 性质 T/F 多选）

Winter 2026 Final · Problem 11.1（regression 性质必选）

Hemanth fits predicted daily_food_lb = 0.04 · weight_lb + 2，给定 r = 0.8，mean weight = 200，SD weight = 100，SD daily_food = 5。Select all statements that must be true:

  1. About 80% of points fall on or near the regression line.
  2. Some other line could have a lower average squared residual than this line.
  3. No bootstrap resample can reproduce exactly this regression line.
  4. None of the above.
显示答案 & 解析

答案：D（None of the above）

  • (A) ❌ r = 0.8 不代表"80% 数据贴线"。r 是"linear pattern 强度"，不是"百分比"。
  • (B) ❌ 错——课程教的 regression line 就是 least-squares 解，它最小化 mean squared error。没有其他直线能比它的 RMSE / MSE 更低。（Lec 24 详细解释。）
  • (C) ❌ 错。bootstrap resample 是有放回抽样，有可能抽到与原 sample 完全相同的样本（虽然概率极小），那时 regression line 完全一样。所以"绝不可能"过强。

陷阱浓度极高：(B) 看起来"很对"——很多人以为可能有更好的 line。但 regression line 的定义就是 minimum RMSE，所以"some other line has lower MSE" 是错的。（这个细节会在 Section 2 / Section 3 详细讲清。）

真题 7：Wi26 Final Q11 (c) ⭐⭐⭐（单位换算 · 变 / 不变）

Winter 2026 Final · Problem 11.3（把 lb 换 kg 后 slope / intercept / r 的变化）

把 weight 和 daily_food 都从 lb 换成 kg（1 lb = 0.45 kg），重新拟合 predicted daily_food_kg from weight_kg。和原线相比，下面三个量是 increase / decrease / stay same？

  • Slope：
  • Intercept：
  • r：
显示答案 & 解析

答案：

  • Slope：stay the same。因为 x 和 y 同时乘了同一个常数 0.45。
    slope = r · SDy / SDx。新 SDy = 0.45 · 原 SDy，新 SDx = 0.45 · 原 SDx，两个 0.45 抵消 → slope 不变。
  • Intercept：decrease（绝对值减小）。b = mean_y − slope · mean_x。新 mean_y = 0.45 · 原 mean_y、新 mean_x = 0.45 · 原 mean_x → 新 b = 0.45 · (原 mean_y − slope · 原 mean_x) = 0.45 · 原 b。原 b = 2 > 0 → 新 b = 0.9 → decrease。
  • r：stay the same。r 对 x、y 的线性变换不变。

记忆：x、y 同步乘同一常数 c → slope 不变（c 上下抵消）、intercept × c、r 不变。如果只 x 或只 y 换单位 → slope 会改变。

真题 8：Wi26 Final Q11 (d) ⭐⭐⭐（只标准化 x · 影响）

Winter 2026 Final · Problem 11.4（只把 x 转 SU，y 保持原单位）

只把 weight 换成 standard units（除以 SD、减 mean），y = daily_food_lb 不变。新的 regression line 相比原线：

  • Slope：increase / decrease / stay same？
  • Intercept：？
  • r：？
显示答案 & 解析

答案：

  • Slope：increase（因为分母 SD_x 变小：从 100 变成 1）。
    • 新 slope = r · SD_y / SD_x_new = 0.8 · 5 / 1 = 4。原 slope 是 0.04。所以 increase（× 100）。
  • Intercept：increase。
    • 新 mean_x = 0（SU 化后），新 b = mean_y − slope · 0 = mean_y = 0.04 · 200 + 2 = 10。原 b = 2，新 b = 10 → increase。
  • r：stay the same。任何线性变换（标准化是 (x − mean) / SD，包括平移 + 缩放，是线性的）r 都不变。

练习 · Concept Check（识别 association vs correlation）

基于 Lec 23 课件 Concept Check 改编

下面哪个最准确描述这张 scatter plot（一个标准抛物线 y = x²，x 从 −6 到 6）？

  • (A) Association and correlation.
  • (B) Association but not correlation.
  • (C) Correlation but not association.
  • (D) Neither association nor correlation.
显示答案 & 解析

答案：(B) Association but not correlation。

  • 明显有 association：知道 x 就能算出 y。
  • 但 r ≈ 0（"correlation" 课程里专指linear，对称抛物线 r 为 0）→ no linear correlation。

(C) 不可能：correlation 是 association 的特殊（linear）形式，r ≠ 0 必然存在 association。
