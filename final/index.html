🔑 必背 12 件套（Final 视角 · 贴在桌前）

  1. Bootstrap loop 模板：resample = sample.sample(sample.shape[0], replace=True)（Lec 14）
  2. 95% CLT CI：[mean − 2·SD/√n, mean + 2·SD/√n]（Lec 17）
  3. z 反推：cdf(z) = (1 + c) / 2（c 是 confidence）
  4. Chebyshev：z=2 → 75%，z=3 → 88.8%，z=4 → 93.75%（适用任何分布）
  5. Normal：68 / 95 / 99.7（仅 normal）
  6. Sample size 公式：n ≥ (4·SD / w)²；0/1 数据 SD 上界 = 0.5
  7. CI 解读：唯一对的版本是"X% of intervals from this process contain truth"
  8. p-value：under null 看到 observed 或更极端的概率。永不 "accept" null
  9. TVD：np.abs(d1-d2).sum() / 2，仅 categorical
  10. Correlation r：(x_su * y_su).mean()，r ∈ [−1, 1]，对线性变换不变，只衡量 linear
  11. Regression line in SU：predicted y_su = r · x_su，过 (0, 0)，slope = r
  12. Regression line in original units：m = r · SD_y / SD_x，b = mean_y − m · mean_x，预测公式 predicted y = m · x + b

⚠️ Top 8 高频陷阱（Final 视角 · 贴脑门上）

  1. "95% chance true mean in CI" ❌：CI 解读高频错答（应是 process / intervals 的语义）
  2. "95% sample means within 2 pop SDs" 答 Yes：官方按 "2 SDs of sample-mean distribution = 2 SEs" 解读（Wi26 Q1.2 陷阱，详见 quiz3 Section 4）
  3. r = 0 ≠ no association：可能是非线性（parabola 反例）
  4. r 对线性变换不变：换 lb ↔ kg、dollars ↔ yen 都不变 r；但 mean、SD、slope（在 original units）会变
  5. Permutation 不是 bootstrap：permutation = shuffle labels（不放回），bootstrap = resample with replacement
  6. TVD 只能用于 categorical：smoking & birth weight 是数值的 → 不能用 TVD
  7. "Accept" the null 永远是错的：只能 "fail to reject"
  8. Single outlier 可以让 r 从 0 变 1：所有 regression 题目第一步都该问"有 outlier 吗？"
