📚 知识点

复习：Regression line in standard units

Section 1 已建立：当 x、y 都在 standard units 时，

predicted y(su)  =  r · x(su)
  • 这条线过原点 (0, 0)，斜率 = r，且 |r| ≤ 1。
  • 用起来不方便：每次预测都要先把 x 转 SU，预测出 y_su 后再转回原单位。

本节目标：把这条线表达成 original units 下的 y = m·x + b 形式，这样直接代入原单位的 x 就能得到原单位的 predicted y。

Regression line in original units（推导）

从 SU 形式出发：

( predicted y − mean of y ) / SD of y  =  r · ( x − mean of x ) / SD of x

把左边乘 SD of y、加 mean of y：

predicted y  =  r · (SDy / SDx) · (x − meanx) + meany

展开成 predicted y = m·x + b：

m  =  r · ( SD of y / SD of x )
b  =  mean of y − m · mean of x

这两条公式是 Final 最高频的考点。题目给"5 个数"（r、mean_x、mean_y、SD_x、SD_y）就能让你算 m 和 b 然后做任意预测；或者反过来给你 m、b 和部分参数让你反推 r 或某个 SD。

Slope & intercept 公式 + 代码

def slope(df, x, y):
    """斜率 m = r · SD_y / SD_x"""
    r = calculate_r(df, x, y)
    return r * np.std(df.get(y)) / np.std(df.get(x))

def intercept(df, x, y):
    """截距 b = mean_y − m · mean_x"""
    return df.get(y).mean() - slope(df, x, y) * df.get(x).mean()

一个关键不变量：regression line 必过 (mean_x, mean_y)

验证：代入 x = mean_x → predicted y = m·mean_x + b = m·mean_x + (mean_y − m·mean_x) = mean_y。✓

⚠️ 这条性质 Wi23 Q7 P3.4 直接考：题目说"原数据集 regression line 过 (500, 15)"——直接告诉你 mean_x = 500、mean_y = 15。这是反推 outlier 位置的关键线索。

用 regression line 做预测

m = slope(mom_son, 'mom', 'son')        # 0.365
b = intercept(mom_son, 'mom', 'son')    # 45.858

def predict_son(mom):
    return m * mom + b

predict_son(62)   # 68.49 inches
predict_son(73)   # 72.51 inches

Galton's data: predicted son's height = 0.365 · mother's height + 45.858

单位 / 线性变换对 slope & intercept 的影响

Final 最高频的"什么变什么不变"题。背熟下面这个表（用 m = r · SD_y / SD_x 和 b = mean_y − m·mean_x 推一下，所有结论都能现场推出来）：

变换rm (slope)b (intercept)解释
x → x + c
(只给 x 加常数)		 不变 不变 −m·c scatter plot 平移不改变形状或斜率；intercept 跟着平移（mean_x 变了）
y → y + c 不变 不变 +c 整体抬升 c，斜率没变，截距 +c
x → c · x
(c > 0)		 不变 ÷ c 不变 SD_x → c·SD_x，分母变 c 倍；mean_x 也变 c 倍，跟新 m 乘起来还是原来的
y → c · y
(c > 0)		 不变 × c × c SD_y → c·SD_y → m 变 c 倍；mean_y 也变 c 倍 → b 变 c 倍
同时 x → c·x、y → c·y
(同一常数)		 不变 不变 × c m: c 上下抵消；b: mean_y → c·mean_y，m 不变，mean_x → c·mean_x → b = c·b
x → (x − mean_x) / SD_x
(只把 x 标准化)		 不变 × SD_x
(变成 r · SD_y)		 变（新 mean_x = 0 → 新 b = mean_y） 分母变成 1；新 b 就是 mean_y
x 和 y 都标准化 不变 = r = 0 这就是 SU 形式（过原点，slope = r）

记忆口诀：

  • "对 x、y 同时加常数"：slope 不变，intercept 跟着调整。
  • "只放缩 y"：slope 和 intercept 都按 y 的缩放因子放大。
  • "只放缩 x"：slope 按 x 缩放因子的倒数（÷ c），intercept 不变。
  • "x、y 同步放缩"：slope 不变（抵消），intercept × c。
  • r 永远对线性变换不变。✓

Outliers 的剧烈影响

课程里的戏剧性例子：8 个干净的数据点 + 1 个 outlier，单个 outlier 把 r 从 0.985 拉到 −0.03，regression line 从陡升变成几乎水平：

数据rregression line
8 干净点 + 1 outlier（位置在左上角）−0.03y = −0.05x + 67.82（几乎水平）
移除 outlier 后0.985y = 0.98x + 3.04（陡升 + 极贴线）

🚨 这是 regression 分析的核心警告：永远先画 scatter plot看 outliers。一两个极端值就能让 r 和 regression line 完全失真。
Final 考题措辞举例："removing the outlier"、"if we added a point at ..."、"the regression line passes through (mean_x, mean_y) and ..."——都在暗示 outlier 的杠杆作用。

Outlier 的"杠杆点"性质

因为 regression line 必过 (mean_x, mean_y)，所以加 / 减一个数据点：

  • 会改变 mean_x、mean_y（pivot 点本身移动）
  • 会改变 SD_x、SD_y（spread 增大或减小）
  • 会改变 r（pivot 周围的"贴线度"变了）
  • → slope 和 intercept 都跟着变

Errors & residuals

error  =  actual y − predicted y

对每个数据点 (x_i, y_i)：

  • actual y = y_i（真实观测值）
  • predicted y = m · x_i + b（regression line 给的预测）
  • error / residual = y_i − (m · x_i + b)

使用 regression line 时，error 也叫 residual（残差）。两个词在regression 语境下是同义词。

error 的两个关键性质（对 regression line）

  • errors 的均值 = 0（理论上精确为 0，浮点上是 1e-15 量级）。直觉：regression line 过 (mean_x, mean_y)，误差正负互相抵消。
  • "好"的 regression line：errors 应该小且没有 pattern（具体见 Section 3 / Lec 25 residual plot）。

RMSE：定义 + 代码

要量化"errors 多大"——课程用 Root Mean Squared Error (RMSE)：

RMSE  =  √( mean of ( errors )² )

三步算法：

  1. 每个 error 平方（避免正负抵消）
  2. 取所有平方误差的均值
  3. 开方根，恢复 y 的单位
# 算 regression line 的 RMSE
predicted_y = m * without_outlier.get('x') + b
errors = without_outlier.get('y') - predicted_y
rmse = np.sqrt((errors ** 2).mean())   # ≈ 2.20

给定任意 (slope, intercept)，可以算它的 RMSE：

def rmse(slope, intercept):
    true = without_outlier.get('y')
    predicted = slope * without_outlier.get('x') + intercept
    return np.sqrt(((true - predicted) ** 2).mean())

RMSE 与 SD 的相似性：SD 是"数据偏离 mean"的平均距离；RMSE 是"数据偏离 regression line"的平均距离。同样的"平方→均值→开方"模板。

Least squares：regression line 就是"最佳"线

问题：有无限多 (slope, intercept) 组合 → 无限多 RMSE。哪一组 RMSE 最小？

from scipy.optimize import minimize   # 课程封装的 minimize

# 最小化 rmse(slope, intercept) 的输入
smallest_rmse_line = minimize(rmse)
# array([0.98, 3.04])

这两个数字看着眼熟吗？

slope(without_outlier, 'x', 'y'), intercept(without_outlier, 'x', 'y')
# (0.9759..., 3.0423...)
🎯 课程核心结论：用 m = r · SD_y / SD_x 和 b = mean_y − m·mean_x 算出的 regression line，就是所有可能的 line 中 RMSE 最小的那条。

等价名字（考试可能换着叫，认清）

  • Regression line
  • Line of best fit（最佳拟合线）
  • Least squares line（最小二乘法线）

找这条 slope / intercept 的方法叫 method of least squares（最小二乘法）。

⚠️ Wi26 Final Q11 (a) 的陷阱：选项 "Some other line could have a lower average squared residual than this line" —— 错。Regression line 的定义就是 minimize squared error，没有其它直线能做到更低。

Quality of fit：什么时候 regression 才有意义？

关键警告：slope / intercept 公式对任何数据都能算出来，但 regression line 本身只在数据"大致 linear"时才有意义。

反例：非线性数据

对一个抛物线状数据集，强行算 regression line：

  • slope ≈ 0.14、intercept ≈ 13.10
  • 这条线看起来几乎水平 + 远离绝大部分数据点
  • "line of best fit" 没错（在所有直线里它确实最优），但直线根本不适合这个数据——应该用更复杂的非线性模型

判断"该不该用 regression line"的三件套：

  • (1) 看 scatter plot 形状是不是 roughly linear
  • (2) 看 |r| 是否接近 1（弱关联 → regression 的预测也弱）
  • (3) 看 residual plot（Lec 25 / Section 3）—— curvature 暴露非线性

⚠️ 常见陷阱与考点

陷阱 1：slope 公式方向写反

正确：m = r · SD_y / SD_x。
错：m = r · SD_x / SD_y（很多人记混）。
记忆法：predicted y 在分子上 → SD_y 在分子上。"y 的变化幅度 / x 的变化幅度" 就是 slope。

陷阱 2：intercept 公式记混

正确：b = mean_y − m · mean_x。
这是 "regression line 必过 (mean_x, mean_y)" 的代数等价。
记忆法：把 (mean_x, mean_y) 代回 y = m·x + b → mean_y = m·mean_x + b → b = mean_y − m·mean_x。

陷阱 3：slope in original units 没有 [-1, 1] 限制

只有 SU 形式下 slope = r 才在 [-1, 1]。original units 下 slope 可以是任意实数。
考题原话："The slope of the regression line, when both variables are measured in original units, is never more than 1." → False。
反例：x 增 1，y 增 20 → slope = 20。

陷阱 4：r > 0 时 slope > 0；r < 0 时 slope < 0（永远）

这是对的，因为 SD 永远 ≥ 0，m = r · SD_y / SD_x 的符号 = r 的符号。
Wi23 Q8 P14.1 直接考：知道 prediction 是 positive 在 SU 上 + x_su 也是 positive → r > 0。

陷阱 5：知道 (slope, intercept) 反推 r、mean、SD

给 m、b 和部分 (mean_x, mean_y, SD_x, SD_y, r) → 用 m = r · SD_y / SD_x 和 b = mean_y − m·mean_x 两条公式联立求解。
注意：这两条公式有 5 个未知数 (r、mean_x、mean_y、SD_x、SD_y)，要求出全部需要至少 3 个额外约束。考题一般会给你足够信息。

陷阱 6：bootstrap regression line ≠ bootstrap prediction

"95% bootstrapped slopes 在 [a, b]、95% bootstrapped intercepts 在 [c, d]" 不等于 "95% predicted prices for x = 500 在 [500a + c, 500b + d]"。
原因：slope 和 intercept 的 bootstrap 分布不独立——同一次 resample 同时决定一组 (m, b)；不能把 slope CI 的端点和 intercept CI 的端点独立组合。
对每个 x 算 prediction CI 的正确做法：对每次 bootstrap 算出 (m_i, b_i)，再算 predicted y_i = m_i · x + b_i，对这些 predicted y 取百分位数。

陷阱 7：dataset 重复行影响 regression

"原 dataset 每个 unique book 一行；如果改成每 copy 一行（有多本相同书）" → regression line 会变吗？会变。因为重复行影响 mean、SD、r（高频出现的书的影响被放大）。
这是 Wi23 Q7 P3.5 直接考。

陷阱 8："y-intercept must be positive since all y values are positive" → False

常见错觉：所有数据点 y > 0，所以 line 在 y 轴上的截距也 > 0。
反例：数据点都在 x > 0 区域且 y > 0，但 regression line向左延伸到 x = 0时 y 可以是负的。intercept 只是数学上 line 在 x = 0 的值，跟数据范围无关。
Wi24 Quiz 6 Q2 P16.4 直接考这个。

真题 1：Wi23 Final Q7 P3.1 ⭐⭐⭐（regression line 多选 · 综合）

Winter 2023 Final · Problem 3.1（Bill's Book Bonanza · 4 个 T/F）

书店 regression line 过 (500, 15)（mean pages = 500，mean price = $15）。其它参数：r = 0.6，SD pages = 200，SD price = 4。下面哪些必为真？多选。

  1. It goes through the point (500, 15).
  2. It goes through the point (200, 4).
  3. Its slope is equal to 0.6.
  4. Its y-intercept is equal to 9.
  5. Its RMSE is larger than the RMSE of some other line through this data.
显示答案 & 解析

答案：A 和 D（官方：Options 1 and 4）

  • (A) ✅ regression line 必过 (mean_x, mean_y) = (500, 15)。这是定义。
  • (B) ❌ 没有理由必过 (SD_x, SD_y)。
  • (C) ❌ slope = r · SD_y / SD_x = 0.6 · 4 / 200 = 0.012，不是 0.6。slope ≠ r（除非 SD 比例为 1）。
  • (D) ✅ b = mean_y − m·mean_x = 15 − 0.012·500 = 15 − 6 = 9。
  • (E) ❌ regression line就是 RMSE 最小的线。没有更小的 RMSE。

套路总结：给 r、mean_x、mean_y、SD_x、SD_y 五个数 → 立刻算 m = r·SD_y/SD_x，然后 b = mean_y − m·mean_x。再把任意 x 代进 m·x + b 做预测。

真题 2：Wi23 Final Q7 P3.2 ⭐⭐（用 slope 算 Δy）

Winter 2023 Final · Problem 3.2（书店 · 同样参数）

"The Martian" 比 "The Simple Wild" 多 30 页。两本书都按 regression line 定价。多多少钱？给出 dollars and cents 数字。

显示答案 & 解析

答案：$0.36

predicted y = 0.012 · x + 9。两本书：

  • Δx = 30 (pages)
  • Δ predicted y = m · Δx = 0.012 · 30 = $0.36

速算技巧：算"两个 x 的 predicted y 之差"只需要 slope，不需要 intercept（intercept 抵消）。

真题 4：Wi23 Final Q7 P3.4 ⭐⭐⭐（outlier 反推 · 杠杆点）

Winter 2023 Final · Problem 3.4（书店 · regression line 被 outlier 影响）

原 Dhruv 用错误数据建了线（其中 "Roadside Picnic" 的 price 误录为 −$12 而非 $12）。改正后重建线，slope 减小。问 "Roadside Picnic" 有多少页？(选项)

  • fewer than 500 pages
  • more than 500 pages
  • exactly 500 pages
  • not enough information
显示答案 & 解析

答案：fewer than 500 pages

关键直觉：

  • regression line 必过 (mean_x, mean_y) = (500, 15)（pivot 点）。
  • 原数据里 "Roadside Picnic" price 是 −$12（极低，离 mean 很远在下方）。
  • 改正后 price 变成 +$12（仍低于 mean = 15，但接近多了）。
  • 这相当于把"原来的 outlier"朝上提了。
  • 这本书在 pivot 的哪一侧（pages 比 500 大还是小）决定了它怎么"撬动"线：
    • 如果 pages < 500（pivot 左侧）：把它的 y 提高 → 左侧 y 上升 → 线 左端上升，右端下降 → slope 减小 ✓
    • 如果 pages > 500：把它的 y 提高 → 右侧 y 上升 → slope 增大 ✗
  • 题目说 slope 减小 → "Roadside Picnic" 在左侧 → pages < 500。

核心套路：outlier 是"杠杆点"，影响方向取决于它在 pivot (mean_x, mean_y) 的哪一侧 + 它的 y 向哪边偏。

真题 5：Wi23 Final Q7 P3.6 ⭐⭐（bootstrap regression · prediction interval 陷阱）

Winter 2023 Final · Problem 3.6（bootstrap slope & intercept 与 prediction interval）

对 scatter plot bootstrap 10000 次，每次得一组 (slope, intercept)。95% 的 slopes 在 [a, b]，95% 的 intercepts 在 [c, d]。这是否意味着 "用 regression line 预测 500 页的书的价格"的 95% 区间是 [500a + c, 500b + d]？

显示答案 & 解析

答案：No

关键：slope 和 intercept 的 bootstrap 分布不独立——同一次 resample 决定一组 (m_i, b_i)。把 slope CI 的端点 a、b 与 intercept CI 的端点 c、d 独立组合是错的。

正确做法：对每次 bootstrap 算 predicted price = m_i · 500 + b_i，得到 10000 个 predicted prices，取这些 prices 的 [2.5, 97.5] percentile。

这个题型 Lec 25（Section 3）会更详细讲。

真题 6：Wi23 Final Q7 P11-13 ⭐⭐⭐（选数据科学工具 · Final 综合考察）

Winter 2023 Final · Problem 11/12/13（每题选最合适的工具）

选项：hypothesis testing / permutation (A/B) testing / Central Limit Theorem / regression

(P11) Are nonfiction books longer than fiction books?
(P12) Do people have more friends as they get older?
(P13) Does an ice cream shop sell more chocolate or vanilla?

显示答案 & 解析
  • P11 → permutation testing。两组（fiction / nonfiction）来自不同 distribution 吗？没有 known population → 不能 standard hypothesis；不是 single proportion → 不是 CLT；两组比较 → permutation。
  • P12 → regression。两个 continuous 变量（age vs # friends）的关系 → regression 最适合。
  • P13 → hypothesis testing。Single proportion（chocolate 占总和的比例 = 0.5 吗）→ standard hypothesis test。

工具判断决策树：

  • 两个 continuous 变量的关系 → regression
  • 两组样本是否同分布（没有 population 参数）→ permutation
  • 单样本 vs 已知 population distribution → standard hypothesis testing（可结合 CLT）
  • 估 population parameter → bootstrap CI 或 CLT CI

真题 7：Su24 Final Q9 Problem 9 ⭐（T/F · slope in original units 范围）

Summer 2024 Final · Problem 9（T/F · slope in original units）

True or False: The slope of the regression line, when both variables are measured in original units, is never more than 1.

显示答案 & 解析

答案：False

反例：如果 x 增 1，y 增 20，slope = 20 > 1。slope = r · SD_y / SD_x 在 original units 下可以是任意实数，因为 SD_y / SD_x 没有上界。只有 SU 形式下 slope = r 才在 [-1, 1]。

真题 8：Su24 Final Q9 P8.1 ⭐⭐⭐（反推 SD_y · 经典题型）

Summer 2024 Final · Problem 8.1（Tour de France 阶段时长）

给定 mean stage length (km) = 200，SD = 80，mean time (h) = 6，r = 2/3。Regression 预测 160 km 的阶段需要 5 hours。求 SD_time。

显示答案 & 解析

答案：3 hours

用 SU 公式 y_su = r · x_su：

  • x_su = (160 − 200) / 80 = −0.5
  • predicted y_su = (5 − 6) / SD_y = −1 / SD_y
  • 代入：−1 / SD_y = (2/3) · (−0.5) = −1/3
  • → SD_y = 3

题型模板：题目给"mean_x、SD_x、mean_y、r、一对 (x, predicted y)" → 缺哪个就用 y_su = r · x_su 反推哪个。SD_y 缺 → 反推 SD_y；r 缺 → 反推 r；mean_y 缺 → 反推 mean_y。

真题 9：Su24 Final Q9 P8.3 ⭐⭐（加常数对 slope 的影响）

Summer 2024 Final · Problem 8.3（所有 stage 加 30 min 休息）

每个 Tour de France stage 强制加 30 分钟（0.5 hours）休息。Regression line 的 slope 会怎么变？

  1. Decrease
  2. Stay the same
  3. Increase
  4. Cannot say for sure
显示答案 & 解析

答案：B（Stay the same）

加 30 min 是 y → y + 0.5。整体平移：mean_y 增 0.5，SD_y 不变（spread 不变），r 不变（SU 计算里平移被减掉）。所以 m = r·SD_y/SD_x 不变。

（intercept 会变：b → b + 0.5。）

真题 10：Su24 Final Q9 P8.4 ⭐⭐⭐（反推 r · 极端情况）

Summer 2024 Final · Problem 8.4（同 mean/SD，新 r）

同样的 means、SDs（mean_x = 200，SD_x = 80，mean_y = 6，SD_y = 1.5）。若预测 360 km 的 stage 需要 9 hours，求 r。

显示答案 & 解析

答案：r = 1

用 SU：

  • x_su = (360 − 200) / 80 = 2
  • predicted y_su = (9 − 6) / 1.5 = 2
  • 代入：2 = r · 2 → r = 1

当 x_su 和 predicted y_su 相等时（且都不为 0），r = 1（完美正向 linear）。

真题 11：Wi24 Quiz 6 P16.4 ⭐⭐（intercept & 正负陷阱）

Winter 2024 Quiz 6 · Problem 16.4（"intercept must be positive"）

True or False: Since the number of parking spaces is always a positive integer, the regression line predicting total spaces from S spaces must have a positive y-intercept.

显示答案 & 解析

答案：False

intercept = regression line 在 x = 0 时的 y 值。即使所有数据点的 x、y 都 > 0，line 向左延伸到 x = 0 时 y 可以是负的。

反例：数据点 (10, 20)、(20, 35)、(30, 50)。slope = 1.5，b = mean_y − m·mean_x = 35 − 1.5·20 = 5（正）。但如果数据是 (10, 5)、(20, 8)、(30, 11)，slope = 0.3，b = 8 − 0.3·20 = 2（仍正）。现在数据是 (100, 10)、(200, 20)、(300, 30)：slope = 0.1，b = 20 − 0.1·200 = 0（正）。
但 (100, 10)、(200, 15)、(300, 18)：slope ≈ 0.04，b = mean_y − slope·mean_x = 14.33 − 0.04·200 ≈ 6.3（正）。要 intercept < 0 比较困难但原理上可能，只要 mean_y < m·mean_x 即可。

关键：intercept 是纯数学，跟数据是否非负无关。

真题 12：Sp23 Final Q11 P6.1/6.2 ⭐⭐（比较 slope 大小）

Spring 2023 Final · Problem 6.1/6.2（4 个 slope 的比较）

4 个 regression 配对：

  1. Predict Rent (mean $2500, SD $500) from Sqft (mean 800, SD 200)，r = 0.7
  2. Predict Sqft from Rent，r = 0.7
  3. Predict Rent from Bedrooms (mean 2, SD 0.5)，r = 0.5
  4. Predict Bedrooms from Rent，r = 0.5

(a) 哪个 slope 最大？(b) 哪个 slope 最小？

显示答案 & 解析

(a) 答案：Option 3（最大）。(b) 答案：Option 4（最小）。

  • Option 1：m = 0.7 · 500 / 200 = 1.75
  • Option 2：m = 0.7 · 200 / 500 = 0.28
  • Option 3：m = 0.5 · 500 / 0.5 = 500（每多一间卧室预测多 $500 房租）
  • Option 4：m = 0.5 · 0.5 / 500 = 0.0005（每多 $1 房租预测多 0.0005 间卧室）

直觉：predict y 的 SD大变量从 SD小变量 → slope 大；反过来 → slope 小。
