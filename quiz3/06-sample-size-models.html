📚 知识点

1. Polling 问题：要多少样本？

背景：你想估计 UCSD 学生中使用 Duolingo 的比例。你想要：

  • 95% confidence level
  • CI 宽度 ≤ 0.06（如 [0.21, 0.27] 可以，[0.21, 0.28] 不行）

问：要采样多少人？

2. Proportions are Means（关键转换）

每个回答 "yes/no" → 编码 1/0。"yes 的比例" = "1 的比例" = sample 的 mean。

sample = [0, 1, 1, 0, 1]
proportion_yes = 3/5 = sample mean = 0.6
关键：proportions 就是 means → CLT 适用于 proportions，95% CI 公式也用同一个 formula。

3. CI 宽度公式（推导 sample size）

95% CI 是 mean ± 2·SD/√n。所以：

width = right − left = 2 · (2 · SD/√n) = 4 · SD / √n

要 width ≤ w：

4 · SD / √n ≤ w
√n ≥ 4 · SD / w
n ≥ (4 · SD / w)²

4. 问题：你还没采样，不知道 SD！

解决：用 SD 的上界来保证够大。

SD of 0s and 1s 的公式

SD of 0s and 1s = √(prop_0 × prop_1)　or　√(p × (1−p))

这是关于 p 的函数，画出来是一个倒 U：

  • p = 0 或 p = 1 时 SD = 0（数据全相同）
  • p = 0.5 时 SD = √(0.25) = 0.5（最大值）
Maximum SD of 0s and 1s = 0.5

5. Sample Size Formula（必背）

把 SD = 0.5（上界）代入 n ≥ (4·SD/w)²：

n ≥ (4 · 0.5 / w)² = (2 / w)²

例（Lec18）：w = 0.06

n ≥ (4 · 0.5 / 0.06)²
  = (2 / 0.06)²
  = (33.33...)²
  ≈ 1111.11

# 取最小整数：1112 人

Activity（Lec18）：w = 0.03

n ≥ (4 · 0.5 / 0.03)²
  = (66.67)²
  ≈ 4444.44

# 取最小整数：4445 人

注意：w 减半 → n 变 4 倍（精度翻倍要付 4 倍代价）。

6. 推广到任意置信度 c%

把 2 换成 z（c% 的 multiplier）：

n ≥ (2z · SD / w)²　or　对 0/1 数据 n ≥ (2z · 0.5 / w)² = (z / w)²

7. Statistical Models（统计模型）

George E. P. Box 名言：「Essentially, all models are wrong, but some are useful.」

定义：A model is a set of assumptions about how data was generated.

目标：assess the quality of a model —— 给定数据，判断模型是否"解释"了数据中的 pattern。

8. Example: Swain v. Alabama (1965)

  • Alabama Talladega County 的合格陪审员 population 中 26% 是 Black。
  • Robert Swain 的陪审团 panel 100 人中，只有 8 个是 Black。
  • Supreme Court 说 "disparity is small"，驳回上诉。
  • 问：模型 "panel 是从 26% Black 的 population 中随机抽 100 人" 能否解释 observed = 8？

9. 三步法模拟（必背框架）

  1. 假设 model 是真的。
  2. 生成大量样本（如 10000 次），每次计算 statistic（这里是 Black 男人数）。
  3. 看 observed 在 empirical 分布上是否典型。如果罕见 → model 站不住。

10. np.random.multinomial 关键 API

np.random.multinomial(sample_size, pop_distribution)
  • 有放回地采样（独立同分布）。
  • 返回 array，长度 = 类别数，每个元素是该类别被采到的次数。
  • pop_distribution 是概率数组，加起来 = 1。

例（Halloween candy）

# 35 个房子，糖果分布 30% Starburst / 30% SourPatch / 40% Twix
np.random.multinomial(35, [0.3, 0.3, 0.4])
# 例如 array([8, 14, 13])（每次结果不同）

例（Swain Step 1：单次模拟）

demographics = [0.26, 0.74]   # [Black, non-Black]
np.random.multinomial(100, demographics)
# 例如 array([26, 74])

# 只要 Black 数（第一个分量）
np.random.multinomial(100, demographics)[0]
# 例如 26

例（Swain Step 2：完整模拟 10000 次）

counts = np.array([])
for i in np.arange(10000):
    new_count = np.random.multinomial(100, demographics)[0]
    counts = np.append(counts, new_count)
# counts 是 10000 个模拟中 Black 男的数量

例（Swain Step 3：可视化 + 判断）

画 histogram，标 observed = 8。结果：10000 次模拟中最小值 = 11，没有任何一次 ≤ 8。结论：observed 在该 model 下几乎不可能发生 → model 是 flawed（不是 chance；是 racial bias）。

11. np.random.choice vs np.random.multinomial 区别

np.random.choice(arr, n, replace=)np.random.multinomial(n, [p1,p2,...])
输入具体的元素 array + 大小大小 + 类别概率分布
输出大小为 n 的 array，每个元素是被采的值长度 = 类别数的 array，每个是次数
有/无放回由 replace 参数决定永远有放回
当 population 未知大小？不行（要给 array）✅ 只需概率分布

⚠️ 常见陷阱与考点

陷阱 1 · w 减半 → n 变 4 倍（不是 2 倍）
n ∝ 1/w²，所以精度要求翻倍代价是四倍采样。
陷阱 2 · SD of 0s and 1s 的最大值是 0.5，不是 1
很多人记成 1。实际上 √(p(1−p)) 在 p=0.5 取最大 = √0.25 = 0.5。
陷阱 3 · 公式里 SD 用上界 0.5，不是真实 SD
因为采样前你不知道 p，所以保守地用 SD = 0.5 保证 n 够大。
陷阱 4 · proportion ≠ percentage
公式里 w 是比例（如 0.06），不是 6%。代入时不要把 6% 当成 6 输入。
陷阱 5 · np.random.multinomial 永远有放回
所以它不是 SRS（DSC10 定义 SRS 为无放回）。W26 Q2 已直接考。
陷阱 6 · multinomial 返回 array 不是单值
要拿第一类别的 count，用 ...[0]。如 np.random.multinomial(100, [0.26, 0.74])[0] 才是 Black 男人数。
陷阱 7 · multinomial 的 probabilities 之和必须 = 1
否则报错或行为未定义。
💡 Sample size 题模板：
  1. 识别置信度 c → z（95% → 2）
  2. 识别想要的宽度 w（如 0.06）
  3. 识别数据类型：
    • 0/1（proportion）→ SD 上界 = 0.5 → n ≥ (2z·0.5/w)² = (z/w)²
    • 给了 sample SD → 直接代入 n ≥ (2z·SD/w)²
  4. 取最小整数（向上取整）
💡 Multinomial 模拟模板： 
counts = np.array([])
for i in np.arange(N_REPS):
    sample = np.random.multinomial(SAMPLE_SIZE, [p1, p2, ...])
    stat = sample[INDEX]   # 取关心的类别
    counts = np.append(counts, stat)
# 然后画 histogram，axvline 标 observed

Lec18 Activity（真题改编）· Sample Size for w=0.03

You want a 95% confidence interval for the proportion of UCSD students who use Duolingo, with width at most 0.03. What's the minimum number of people you need to sample?

显示答案

答案：n = 4445

n ≥ (4 · 0.5 / 0.03)²
  = (2/0.03)²
  = (66.666...)²
  = 4444.44...

# 取最小整数 → 4445
练习 · Sample Size 通用

You want a 95% CI for a proportion with width ≤ 0.1. Minimum sample size?

显示答案

答案：n = 400

n ≥ (4 · 0.5 / 0.1)² = (20)² = 400
练习 · Sample Size 给定 SD

You're estimating mean concert attendance with a 95% CI of width at most 10. The sample SD is approximately 50. Minimum n?

显示答案

答案：n ≥ 400

n ≥ (4 · SD / w)²
  = (4 · 50 / 10)²
  = (20)²
  = 400
练习 · SD of 0s and 1s

A sample of 1000 students has 300 who use Duolingo and 700 who don't. Compute the sample SD.

显示答案

答案：SD ≈ 0.458

p = 0.3
SD = √(p · (1-p)) = √(0.3 · 0.7) = √0.21 ≈ 0.458
Winter 2026 Quiz 3 · Problem 2（multinomial 在 SRS 题中出现）

Select all sampling methods that give an SRS of 20 songs:

  • (B) Use np.random.multinomial(20, np.ones(songs.shape[0]) / songs.shape[0]) to decide how many times to take each row.

(B) 是 SRS 吗？

显示答案

(B) 不是。np.random.multinomial 永远有放回采样，所以同一首歌可能被采多次。DSC10 的 SRS 定义是无放回，所以这不是 SRS。

练习 · Swain 模拟模板

Population 中 60% 是某类。我们要模拟从该 population 抽 200 人，并统计该类别有多少。重复 5000 次。填空：

counts = np.array([])
for i in np.arange(__(a)__):
    new_count = np.random.multinomial(__(b)__, __(c)__)[__(d)__]
    counts = np.append(counts, new_count)
显示答案

(a) 5000　(b) 200　(c) [0.60, 0.40]　(d) 0

练习 · multinomial 输出

What does the following return?

np.random.multinomial(50, [0.2, 0.3, 0.5])
显示答案

形状：array，长度 = 3。内容：每个分量是对应类别被采到的次数。总和必为 50。

例：array([12, 17, 21])（每次随机不同；总和 = 12+17+21 = 50 ✓）

练习 · 模型判断 (Swain 风格)

10000 次模拟下的 histogram 显示最小值 = 18，最大值 = 41，中心 ≈ 26。如果 observed = 24，结论？如果 observed = 8，结论？

显示答案
  • observed = 24：落在分布中心附近 → 模型合理，没理由拒绝。
  • observed = 8：远低于所有 10000 次的最小值 18 → 模型下几乎不可能发生 → 模型 flawed（这就是 Swain 的实际情况）。
练习 · w 减半的代价

If width requirement halves from 0.06 to 0.03, by how much does the required sample size grow?

显示答案

4 倍。因为 n ∝ 1/w²。具体：1112 → 4445，约 4 倍。
