📚 知识点

1. Probability distribution vs Empirical distribution

两种"分布"，区别在于来源是理论还是观测。

Probability distributionEmpirical distribution
来源理论 (theoretical)观测 (observations)
描述所有可能取值 + 每个值的理论概率所有观察到的值 + 每个值的出现频率/比例
例掷骰子，每面 1/6掷25次骰子，得到 array([4,1,6,...,3,4,4])，再统计每面频率

Law of large numbers（大数定律）：如果一个 chance experiment 在相同条件下独立重复多次，那么某事件发生的比例会越来越接近其理论概率。

📌 大白话：掷骰子次数越多，得到 5 的频率越接近 1/6。这就是我们能用 simulation 近似 probability distribution 的依据。

2. Population vs Sample

  • Population（总体）：我们想了解的全部人/物/事件。
  • Sample（样本）：population 的子集。通常我们没法调查全体，只能调查子集。
  • 目标：用 sample distribution 估计 population distribution。

3. Simple Random Sample (SRS)

DSC10 里 SRS 的严格定义（必背！）：
"A simple random sample is a sample drawn uniformly at random without replacement."
  • Uniformly：每个个体被选中的概率都相同。
  • Without replacement：同一个个体不会被选两次。

不好的采样方式

  • Convenience sample：方便谁就采谁（如 PC 排队的人）。容易有 bias（偏倚）。

4. 采样代码：两套语法，默认值正好相反！

从 list / array 采样：np.random.choice

colleges = ['Revelle', 'Muir', 'Marshall', 'Warren',
            'ERC', 'Sixth', 'Seventh', 'Eighth']

# SRS of 3 colleges（必须显式写 replace=False）
np.random.choice(colleges, 3, replace=False)
# array(['Seventh', 'Eighth', 'Muir'])

从 DataFrame 采样：df.sample

# 5 flights, without replacement（default 就是 False）
united_full.sample(5)

# 5 flights, with replacement
united_full.sample(5, replace=True)
🚨 极易错点：两个 API 的 replace 默认值正好相反！
  • np.random.choice(options, n) → 默认 replace=True（有放回）
  • df.sample(n) → 默认 replace=False（无放回）
考试 SRS 题里：np.random.choice(songs.get("song_name"), 20) 因为默认 with replacement，不是 SRS。

5. Parameter vs Statistic

Parameter（参数）Statistic（统计量）
关联populationsample
是否随机固定（fixed, not random）随机（depends on the random sample）
例population meansample mean
实际操作通常未知 / 想求从样本算出来，用来 estimate parameter
记忆口诀：parameter / population 都以 p 开头；statistic / sample 都以 s 开头。

关键："Statistical inference" = 用 sample 数据对 population 下结论。

6. Sampling Distribution of a Statistic（统计量的分布）

因为 statistic 依赖随机样本，所以 statistic 本身也是随机的，有自己的分布。

  • Probability distribution of a statistic（= "sampling distribution"）：理论上所有可能样本下的统计量取值及其概率。
  • Empirical distribution of a statistic：通过模拟大量样本、对每个样本算出统计量、画直方图得到的分布。当模拟次数足够多，它逼近 sampling distribution。

模拟 sampling distribution 的标准模板（必背）

sample_size = 1000
repetitions = 2000
sample_means = np.array([])

for n in np.arange(repetitions):
    m = united.sample(sample_size).get('Delay').mean()
    sample_means = np.append(sample_means, m)

# 把 sample_means 画 histogram → 这就是 empirical distribution
# of the sample mean

7. 样本量对 sampling distribution 的影响

  • Center（中心）：不变，始终在 population mean 附近。
  • Spread（宽度）：样本越大 → 分布越窄（更精确）。

这是 Quiz3 高频 T/F 选项：「Larger sample → better estimate」永远是 True。「Distribution of statistic 与 sample size 无关」永远是 False。

⚠️ 常见陷阱与考点

陷阱 1 · replace 默认值反着来
np.random.choice 默认 replace=True，df.sample 默认 replace=False。任何"判断这段代码是否是 SRS"的题都先看默认值。
陷阱 2 · replace=1 是 truthy
songs.sample(20, replace=1) 等价于 replace=True → with replacement → 不是 SRS。
陷阱 3 · 从 population 采样 ≠ bootstrapping
Bootstrapping 严格定义是从 sample 中 with replacement 再采样。如果数据本身就是 population（如 Fall 2024 Q1.2 的 1000 reptiles），那不管你 replace=True 与否，都不叫 bootstrap。
陷阱 4 · Stratified sampling 不是 SRS
"先随机选 20 个 genre，再每个 genre 选一首歌" —— 这种分层抽样下每个 song 被选中的概率与所在 genre 大小有关，不是 uniform，所以不是 SRS。
陷阱 5 · np.random.multinomial 是 with replacement
np.random.multinomial(20, np.ones(n)/n) 等价于"独立、均匀地、有放回地"抽 20 次 → 不是 SRS。
陷阱 6 · 「Parameter 随样本而变」是 False
Parameter 是 population 的固定数值；变的是 statistic。Sp24 Q1 直接考。
陷阱 7 · 「boot_means 是 distribution of parameters」是 False
模拟出的 histogram 上的每个值都是一个 sample 的统计量（不是 parameter）。
陷阱 8 · 「Empirical distribution 是 theoretical」是 False
Empirical 来自 observations / simulations；theoretical 那个叫 probability distribution。
陷阱 9 · 「sample 比 population 多采样几个会有放回」
df.sample(n, replace=True) 可以让 n > population size；但 df.sample(n)（默认 replace=False）如果 n > len(df) 会报错。
💡 高分技巧：见到任何"分布"字眼，先在脑里贴四个标签：
  1. 什么的分布？（population / sample / statistic）
  2. 理论还是观测？（probability / empirical）
  3. 谁是 parameter，谁是 statistic？
  4. 采样有放回吗？（是否 SRS）

Spring 2024 Quiz 3 · Problem 1（T/F 多选）

Which of the following statements are true in general? Select all that apply.

  • (A) Parameters are fixed, but statistics can change depending on the sample.
  • (B) Parameters and statistics can both fluctuate depending on the sample.
  • (C) For simple random samples, statistics give better estimates of parameters when the sample size is larger.
  • (D) The distribution of a statistic is the same regardless of the sample size.
  • (E) None of the above.
显示答案

答案：(A) 和 (C)

  • (A) ✅ Parameter 是 population 固定量；statistic 因样本而变。
  • (B) ❌ Parameter 不随样本变。
  • (C) ✅ 大数定律 + 抽样直觉：样本大 → 估计准。
  • (D) ❌ Sampling distribution 会随 n 变窄。
Spring 2024 Quiz 3 · Problem 3（SRS 识别 · 高频）

Which of the following can be used to generate a simple random sample of "rating"s from 10 restaurants in restaurants? Select all that apply.

# Option 1
sample = restaurants.take(np.arange(10)).get("rating")

# Option 2
sample = restaurants.sample(10, replace=False).get("rating")

# Option 3
sample = restaurants.sample(10, replace=True).get("rating")

# Option 4
positions = np.random.choice(np.arange(0, restaurants.shape[0]),
                             10, replace=False)
sample = restaurants.take(positions).get("rating")

# Option 5
positions = np.random.choice(np.arange(0, restaurants.shape[0]),
                             10, replace=True)
sample = restaurants.take(positions).get("rating")
显示答案

答案：Option 2 和 Option 4

  • Option 1 ❌ take(np.arange(10)) 取前 10 行，不随机。
  • Option 2 ✅ sample(10, replace=False) 标准 SRS。
  • Option 3 ❌ replace=True 是有放回，不是 DSC10 定义的 SRS。
  • Option 4 ✅ 用 np.random.choice + replace=False 选 position 再 take，等价 SRS。
  • Option 5 ❌ replace=True。
Winter 2026 Quiz 3 · Problem 2（SRS 识别 · 新趋势严格定义）

Select all sampling methods below where the resulting sample is a simple random sample (SRS) of 20 songs from the songs DataFrame.

  • (A) songs.sample(20, replace=1)
  • (B) np.random.multinomial(20, np.ones(songs.shape[0]) / songs.shape[0]) to decide how many times to take each row.
  • (C) np.random.choice(songs.get("song_name"), 20)
  • (D) Randomly pick 20 genres without replacement, then within each picked genre, randomly pick a song.
  • (E) Randomly pick 20 artists without replacement, then within each picked artist, randomly pick a song.
  • (F) None of the above.
显示答案

答案：(F) None of the above

  • (A) ❌ replace=1 是 truthy = True → with replacement，不是 SRS。
  • (B) ❌ np.random.multinomial 等价于有放回均匀抽样。
  • (C) ❌ np.random.choice 默认 replace=True；而且只取了一列 song_name，没在 row 层面采样。
  • (D)(E) ❌ Stratified sampling（分层抽样）：每首歌被选中的概率与所在 genre/artist 大小有关，不 uniform。

这是 W26 最新趋势：考 SRS 严格定义。务必记住 SRS = uniform + without replacement。

Spring 2025 Quiz 3 · Problem 2.1（情景 T/F）

Setup：UCSD 想估计学生愿为 Sun God Festival 票付多少钱。他们采集了 500 个 simple random samples，每个 100 个学生，问相同问题。然后画了一个直方图，展示每个样本的均值的分布。

Which of the following statements are true? Select all that apply.

  • (A) The histogram will be approximately centered around the mean amount that all UCSD students would be willing to pay.
  • (B) The variability in the histogram is due to the fact that we resample with replacement.
  • (C) This distribution is an example of an empirical distribution.
  • (D) This distribution includes many randomly generated parameters.
  • (E) None of the above.
显示答案

答案：(A) 和 (C)

  • (A) ✅ Sampling distribution of sample mean 的中心 ≈ population mean。
  • (B) ❌ 这不是 bootstrap！是从 population 反复取 SRS，根本没用 replacement。
  • (C) ✅ 通过观察/模拟得到的分布 → empirical。
  • (D) ❌ 每个样本的 mean 是 statistic，不是 parameter。
Spring 2025 Quiz 3 · Problem 2.2（一词答）

Describe in one word how the histogram would be different if it were instead based on 500 simple random samples of 1000 students each (instead of 100).

显示答案

答案：Narrower（或 tighter / thinner）

样本量 n 增大 → sampling distribution 的 spread 减小 → 直方图更窄。center 不变。

Winter 2024 Quiz 4 · Problem 3.2（圈词）

承前题：boot_means 由 10000 次 bootstrap resample 得到。

In each blank, circle the word that correctly fills in the sentence.

A histogram of boot_means shows a(n) probability / empirical distribution of a statistic / parameter.

显示答案

答案：empirical / statistic

  • empirical：来自实际跑模拟得到的（不是理论）。
  • statistic：每次 resample 算的 mean 是 statistic（parameter 是固定数）。
Winter 2025 Quiz 3 · Problem 1（5 词填空 · 极高频）

A (a) provides insight into how much a (b) might vary across samples and is key for estimating the corresponding (c). This type of distribution is an example of a (d) because it is based on trials of an experiment, as opposed to a (e), which is based on theory.

五个词：parameter, statistic, probability distribution, empirical distribution, bootstrap distribution

显示答案
  • (a) bootstrap distribution — 反映 statistic 在不同 sample 下的变化。
  • (b) statistic — 会随样本变的量。
  • (c) parameter — 我们想估计的固定 population 数值。
  • (d) empirical distribution — based on trials of an experiment。
  • (e) probability distribution — based on theory。
Fall 2024 Quiz 3 · Problem 1.1（语法填空）

space_reptiles 是 1000 行的 population（所有 Statistica 上的爬行动物），列有 length、age、number_of_eyes。

Fill in the blanks. The function takes a sample of reptiles from space_reptiles, with replacement, of the specified size, and return the average value in the given column for the sample.

def sample_of_reptiles(sample_size, column):
    return space_reptiles.sample(__(x)__).__(y)__
显示答案

(x) sample_size, replace=True

(y) get(column).mean()

完整：space_reptiles.sample(sample_size, replace=True).get(column).mean()

注意 column 是变量（字符串值由参数传入），所以 get(column) 不加引号。

Fall 2024 Quiz 3 · Problem 1.2（T/F · 概念辨析）

True or False: The function call sample_of_reptiles(1000, "length") is an example of bootstrapping.

显示答案

答案：False

Bootstrapping 严格定义是从 sample 中再 with replacement 采样来估计 statistic 的变异性。这里 space_reptiles 是整个 population，不是 sample。即便用 replace=True 而且 size 等于 population size，这也只是普通的 population sampling，不是 bootstrap。
