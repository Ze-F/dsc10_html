📚 知识点

1. 为什么需要 Bootstrap？

问题：我们有一个 sample（如 500 个员工工资），算出 sample statistic（如 sample median）。但 statistic 是随机的——换一个 sample 答案会不同。我们想知道"如果换一个 sample，估计值能差多远"。

理想做法：从 population 反复抽 sample → 算很多次 statistic → 看分布。
但：现实里我们只有一个 sample，没法回到 population 再抽。

💡 关键洞察：sample 看起来很像 population。所以我们可以把 sample 当作 population，从 sample 里 with replacement 再抽 sample——这就是 bootstrapping。

2. Bootstrap 的精确定义

Bootstrapping = 从 sample（不是 population！）中 with replacement 抽取与原 sample 同样大小的 resample。

每个 resample 算一次 statistic，重复 N 次（通常 5000-10000），得到 statistic 的经验分布，叫 bootstrap distribution。

3. 为什么必须 with replacement？

如果 without replacement，每次 resample 都会选出 sample 里的所有元素（顺序可能不同），但内容一模一样 → statistic 永远不变 → 完全没用。

# Without replacement（错误示范）
# 原 sample = [7, 9, 4]，每次都返回 [7,9,4] 的某个排列
# 三个数的 median 永远 = 7

# With replacement（正确）
# 可能得到 [9,4,9] → median = 9
# 可能得到 [4,4,9] → median = 4
# 可能得到 [7,4,7] → median = 7
# 不同 resample 给不同 statistic ✓
🤔 常见困惑：「抽 n 次，不就会把 sample 里每个人都抽一遍吗？只有一种结果？」

不会。关键在「with replacement」——每次抽都是独立地从原 sample 里随便挑一个，所以同一个 individual 可以被抽多次，也可能一次都没被抽到。

原 sample = [A, B, C]（n=3），抽 3 个：可能 [A,A,A] / [A,B,B] / [C,A,C] / [A,B,C] / [B,B,A] / ... 总共 3³ = 27 种。

「每个 individual 恰好被抽一次」的概率 = n! / nⁿ：
  • n=3：6/27 ≈ 22%
  • n=10：≈ 0.036%
  • n=100：≈ 10⁻⁴²（几乎不可能）

所以当 sample 不太小（n ≥ 几十），几乎每次 resample 都和原 sample 不同——有些个体重复、有些缺席。这正是 bootstrap 能模拟「从 population 反复抽 sample」的关键：差异性来自 with replacement，而不是来自"抽得不全"。

4. Bootstrap 标准模板（必背！）

np.random.seed(38)  # 老师代码里常见，考试不要求

n_resamples = 5000
boot_stats = np.array([])

for i in np.arange(n_resamples):
    # 1. Resample from my_sample WITH REPLACEMENT, same size
    resample = my_sample.sample(my_sample.shape[0], replace=True)

    # 2. Compute statistic on this resample
    stat = resample.get('column').median()  # 或 mean / 其他

    # 3. Store it
    boot_stats = np.append(boot_stats, stat)
三个必背要点：
  1. .sample(n, replace=True) 的 n 必须是 原 sample 大小（通常写 my_sample.shape[0] 或题目给的 sample_size）
  2. replace=True 不能漏
  3. 累加用 boot_stats = np.append(boot_stats, stat)——必须重新赋值

5. Percentiles（百分位数）

定义：pth percentile 是一个数，≥ 数据中至少 p% 的值。

np.percentile(array, p)  # array 不需要事先排序

np.percentile([4, 6, 9, 2, 7], 50)  # 6.0（即 median）
np.percentile([2, 4, 6, 7, 9], 50)  # 6.0（排过序结果一样）

6. 构造 c% Confidence Interval（CI）

步骤：

  1. 把 boot_stats 的中间 c% 取出来。
  2. 左右两端各砍掉 (100 − c) / 2 百分点。
# 95% CI（最常见）
left  = np.percentile(boot_stats, 2.5)
right = np.percentile(boot_stats, 97.5)
ci = [left, right]

各置信度对应的百分位（必背）

Confidence c%每端砍掉 (100−c)/2 %左 percentile右 percentile
99%0.5%0.599.5
95%2.5%2.597.5
92%4%496
90%5%595
88%6%694
85%7.5%7.592.5
80%10%1090
50%25%2575
口诀：左 percentile = (100−c)/2，右 percentile = 100 − 左 = (100+c)/2。

7. Bootstrap distribution 近似的是什么？

这是极高频概念题（W26 Q3.4 直接考）。答案：

Bootstrap distribution of a statistic ≈ sampling distribution / probability distribution of that statistic

它不是：

  • ❌ population distribution（不是单个 value 的分布）
  • ❌ probability distribution of population（同上）
  • ❌ distribution of parameters（parameter 是固定数，没有分布）

8. 完整流程示例（Lec14 San Diego salary）

# Setup: we have ONE sample of 500 employees
my_sample = population.sample(500)
sample_median = my_sample.get('TotalWages').median()  # 82508.0

# Bootstrap
np.random.seed(38)
n_resamples = 5000
boot_medians = np.array([])
for i in range(n_resamples):
    resample = my_sample.sample(500, replace=True)
    median = resample.get('TotalWages').median()
    boot_medians = np.append(boot_medians, median)

# 95% CI for population median
left  = np.percentile(boot_medians, 2.5)   # 70671.5
right = np.percentile(boot_medians, 97.5)  # 86405.0
ci = [left, right]

# 解读: "A 95% confidence interval for the population median
#         is $70,671.50 to $86,405."

⚠️ 常见陷阱与考点

陷阱 1 · 漏 replace=True
Bootstrap 必须 with replacement。漏掉 = 每个 resample 都是原 sample 的排列 = statistic 永远不变。
陷阱 2 · resample 大小错
必须等于原 sample 大小（不是 population 大小，不是 100）。若题目说 sample 有 500 个，sample(500, replace=True)。常见错误是写 .sample(1000, ...)。
陷阱 3 · Percentile 算错
90% CI 不是 np.percentile(arr, 10) 和 np.percentile(arr, 90)——其实就是 5/95！记忆口诀：(100−c)/2 和 (100+c)/2。
✅ 90% → 5, 95
✅ 80% → 10, 90（注意：80% 才是 10/90）
✅ 85% → 7.5, 92.5
✅ 88% → 6, 94
✅ 92% → 4, 96
陷阱 4 · 从 population 采样不叫 bootstrap
F24 Q1.2 直接考。Bootstrap 严格意义是从 sample 中再采样。如果你"有 population 的所有数据"还做 .sample(n, replace=True)，那也只是普通采样，不是 bootstrap。
陷阱 5 · 累加忘记重新赋值
np.append(boot_stats, stat) 不会修改原数组；必须 boot_stats = np.append(boot_stats, stat)。
陷阱 6 · "bootstrap distribution 是 distribution of population values" — False
Boot 分布是 statistic 的分布，不是 individual values 的分布。
陷阱 7 · "boot_means 是 distribution of parameters" — False
Parameter 是固定数，没有分布；分布的是统计量。
陷阱 8 · CI 宽度与置信度的关系
更高置信度 → 区间更宽（包住更多）
更低置信度 → 区间更窄
举例: 99% CI > 95% CI > 80% CI（同一份数据）
💡 关于解读 CI 的陷阱，详见 Section 3。Section 2 重点是构造 CI（找 percentile），Section 3 是解读 CI（避开那些"X% chance"的错误说法）。

📝 往年真题

Fall 2025 Quiz 3 · Problem 1.1（Bootstrap 填空 · 含 subset）

Setup：cats 是 sample，含 name, breed, treats, sleepy。我们想构造 sample 中"获得 ≥10 个 treat 的 sleepy 猫的比例"的 bootstrap 分布。

n_resamples = 5000
boot_props = np.array([])
for i in np.arange(n_resamples):
    resample = cats.sample(__(a)__, replace=__(b)__)
    subset = resample[__(c)__]
    prop = subset.shape[0] / resample.shape[0]
    boot_props = np.append(boot_props, prop)
显示答案

(a) cats.shape[0]（resample 必须等于原 sample 大小）

(b) True

(c) (resample.get('sleepy')) & (resample.get('treats') >= 10)（注意用 &、用 resample 不是 cats，每个条件加括号）

Fall 2025 Quiz 3 · Problem 1.2（90% CI percentile）

承上：要从 boot_props 构造一个 90% bootstrap confidence interval，填空：

left  = np.percentile(boot_props, __(a)__)
right = np.percentile(boot_props, __(b)__)
显示答案

(a) 5　(b) 95

90% CI: 砍掉两端各 5%。

Spring 2024 Quiz 3 · Problem 2.1（Bootstrap 函数填空）

函数 boot_meal_price 应该返回意大利餐厅 meal_price 的 bootstrap sample mean（一个值）。restaurants 是 sample，含 cuisine 列。

def boot_meal_price():
    italian = __(a)__
    return italian.sample(__(b)__, __(c)__).get(__(d)__).mean()
显示答案

(a) restaurants[restaurants.get('cuisine') == 'Italian']

(b) italian.shape[0]

(c) replace=True

(d) 'meal_price'

常考组合：先按某条件过滤出 subset，再对 subset 做 bootstrap。注意 resample size = subset 大小，不是原 DataFrame 大小！

Spring 2024 Quiz 3 · Problem 2.2（88% CI · 不常见置信度）

跑 boot_meal_price 10000 次得到 array boot_means。构造一个 88% confidence interval：

left  = np.percentile(boot_means, __(a)__)
right = np.percentile(boot_means, __(b)__)
显示答案

(a) 6　(b) 94

88% CI: (100−88)/2 = 6%，所以左 6、右 94。

def estimate_attendance():

Winter 2025 Quiz 3 · Problem 2.1（SRS + Bootstrap 95% CI）

从 geisel DataFrame（含所有 Geisel 学生）取 SRS of 50，然后构造 95% CI for mean study hours。填空：

my_sample = geisel.sample(__(a)__)
n_resamples = 10000
boot_means = np.array([])
for i in np.arange(n_resamples):
    resample = my_sample.sample(__(b)__, replace=__(c)__)
    boot_means = np.append(boot_means, resample.get('hours').mean())
ci = [np.percentile(boot_means, __(d)__),
      np.percentile(boot_means, __(e)__)]
显示答案

(a) 50　(b) 50　(c) True　(d) 2.5　(e) 97.5

Winter 2025 Quiz 3 · Problem 2.2（角色辨析）

在 bootstrap 代码里：

for i in np.arange(n_resamples):
    resample = my_sample.sample(50, replace=True)
    x = resample.get('hours').mean()
    boot_means = np.append(boot_means, x)
# y = histogram of boot_means

(a) x 是什么？　(b) y（boot_means 的 histogram）是什么？

显示答案

(a) x：a statistic computed from a single resample（这是 sample mean of one bootstrap resample）。

(b) y：an empirical distribution of the sample mean（即 bootstrap distribution），近似 sampling/probability distribution of the sample mean。

Winter 2026 Quiz 3 · Problem 3.1（Bootstrap resample 填空）

对 ray_playlist（Ray 的 sample of songs）做 bootstrap：

resample = ray_playlist.sample(__(a)__, replace=True)
显示答案

(a) ray_playlist.shape[0]

不能写常数（题没告诉你大小）；必须用 .shape[0] 表达"原 sample 大小"。

Winter 2026 Quiz 3 · Problem 3.2（80% CI · 反直觉百分位）

从 boot_means 构造一个 80% bootstrap confidence interval：

left  = np.percentile(boot_means, __(a)__)
right = np.percentile(boot_means, __(b)__)
显示答案

(a) 10　(b) 90

反直觉点：很多人误以为 80% CI 是 percentile 20/80。其实砍掉两端各 (100−80)/2 = 10%，所以 10/90。

Winter 2026 Quiz 3 · Problem 3.4（Bootstrap 分布近似的是什么）

After generating 1000 bootstrap means, the distribution of boot_means approximates...

  • (A) the sampling distribution of the sample mean
  • (B) the population distribution
  • (C) the empirical distribution of the population
  • (D) the probability distribution of a single observation
显示答案

答案：(A) the sampling distribution of the sample mean（即 probability distribution of the sample mean）。

不是 population distribution（那是 individual values 的分布，不是 mean 的分布）。

Winter 2024 Quiz 4 · Problem 3.1（SRS + Bootstrap）

从 oscars DataFrame 取 SRS of 400，然后 10000 次 bootstrap sample means of rating：

my_sample = __(x)__
n_resamples = 10000
boot_means = np.array([])
for i in np.arange(n_resamples):
    resample = __(y)__
    mean = __(z)__
    boot_means = np.append(boot_means, mean)
显示答案

(x) oscars.sample(400)（SRS = without replacement，default）

(y) my_sample.sample(400, replace=True)

(z) resample.get('rating').mean()

注意 (x) 不能写 .sample(400, replace=True)——题目要 SRS。

Fall 2024 Quiz 3 · Problem 2.4（直方图对比）

展示两张 overlaid 的 bootstrap mean histograms：一张用 sample size 400，一张用 sample size 900（都从同一 population 取 SRS 后 bootstrap）。问哪张较窄？哪张 center 相同？

显示答案

n=900 的 histogram 更窄（更小 spread）。两张 center 相近（都接近 population mean）。不会 shifted left/right。

原理：sample mean 的 sampling distribution 的 SD = pop_SD / √n（详见 Section 5 CLT）。所以 n 越大 → spread 越小。
