📚 知识点

1. 什么是 95% 置信？—— "Confidence is in the PROCESS"

记一个三步流程 (Lec15 P3)：

  1. 从 population 收集一个 new original sample；
  2. 从该 sample 反复 bootstrap，计算 statistic 的经验分布；
  3. 取中间 95% 构造 95% CI。
✅ 95% confidence level 的唯一正确解读：
"如果我们反复执行上述三步流程产生 CI，约 95% 的此类 CI 会包含真实的 population parameter。"
("Approximately 95% of the intervals we create through this process will contain the true population parameter.")

关键直觉（Lec15 的 ring toss 类比）：你扔了 200 次圈圈套同一个柱子。如果你瞄得准，~95% 的圈套住了柱子。这里"95%"是你整个流程的命中率，不是某一个圈套住的概率。

2. CI 解读的 4 大陷阱 （必背！）

假设我们造出一个 95% CI = [70671.5, 86405]，下面列出每一句的对错与原因：

说法对/错原因
"There is a 95% chance the true parameter is in [70671.5, 86405]." ❌ 区间一旦造出来就是固定的；parameter 也是固定的。要么在里面，要么不在，没有概率。
"95% of the (population/sample) data values lie in this interval." ❌ CI 描述的是parameter 的位置，不是 data values 的分布。例如 CI 可能很窄，但 population 里大部分值都在外面。
"95% of all bootstrap resample medians lie in this interval." ✅ 这是构造方式本身。但要小心——题目里很容易把"sample medians"偷换成"sample values" / "population values"，就变错了。
"If we repeated the (sampling + bootstrap + CI) process many times, about 95% of the resulting intervals would contain the true parameter." ✅ 这是 confidence level 的正式定义。
"This particular interval contains the true parameter." ❌ 不知道。可能包含可能不。
"The interval contains the sample median." ✅ 因为 CI 是从 bootstrap 分布中间 95% 取的，sample median 几乎总在 bootstrap 分布中间，所以一般会在。但要小心——有些边缘情况下 sample median 可能不在 CI 里（如有偏 bootstrap 分布）。题目通常视为对。

3. Confidence Tradeoffs（置信度与宽度的取舍）

  • 更高置信度（99%）→ 区间更宽（更稳但不精确）。
  • 更低置信度（80%）→ 区间更窄（更精确但更可能 miss）。
  • 更大样本（n 增大）→ 区间更窄（sampling distribution 更紧）。
记忆口诀：
Confidence ↑ → Width ↑（同一份数据）
Sample size n ↑ → Width ↓

4. Bootstrap 的局限

  • Sensitive statistics（极值）不能 bootstrap：max / min。因为 bootstrap resample 的最大值不可能超过原 sample 的最大值，所以 boot 分布永远低估 population max。
  • Bootstrap 只有原 sample 代表性好时才有用。如果原 sample biased，bootstrap 出来的也 biased，无法救回来。

5. Center: Mean vs Median

Mean (均值)Median (中位数)
定义balance point（"杠杆中心"）half-way point（一半数据小于它）
数学性质所有数据偏离 mean 的差之和 = 0左右两边数据点个数相等（不是面积！是面积 50%）
对 outlier敏感（一个大值能拉走 mean）robust（不受极值影响）
对称分布mean = median

Skew 方向决定 mean / median 关系

  • 右偏 (right-skewed, long tail on right)：mean > median（mean 被尾巴拉到右边）
  • 左偏 (left-skewed, long tail on left)：mean < median
  • 对称 (symmetric)：mean = median
记忆口诀："Mean 跟着 tail 走"。tail 在右，mean 在右；tail 在左，mean 在左。

6. Spread: Variance & Standard Deviation

核心公式（必背）

deviationi = valuei − mean

variance = mean of squared deviations
= [(value1 − mean)² + ... + (valuen − mean)²] / n

SD = √variance

三步走计算（手算题模板）

  1. 算 mean
  2. 每个值减 mean → deviations
  3. 每个 deviation 平方 → 求平均 → variance；开根号 → SD

例：data = [2, 3, 3, 9]

mean = (2+3+3+9)/4 = 4.25
deviations = [2-4.25, 3-4.25, 3-4.25, 9-4.25]
           = [-2.25, -1.25, -1.25, 4.75]
squared    = [5.0625, 1.5625, 1.5625, 22.5625]
variance   = (5.0625+1.5625+1.5625+22.5625)/4 = 7.6875
SD         = sqrt(7.6875) ≈ 2.7726

关键性质

  • np.mean(deviations) = 0，永远（这是 mean 的定义性质）。所以单纯"平均 deviation"没用，要平方。
  • SD 单位与原数据相同（如 dollars）；variance 单位是 原单位的平方（dollars²）。
  • SD 可以是 0（所有值相等），但不能是负数。Variance 同理 ≥ 0。
  • SD 越大 → 数据越分散。
  • "Bulk of data is within a few SDs of the mean" —— 准备 Section 4 学的 Chebyshev/Normal 的铺垫。

函数

np.std(data)   # 返回 SD（注意：分母是 n，不是 n-1）
np.mean(data)
np.median(data)

7. Sum of Deviations 必为 0 —— 概念题

对任何 dataset：

Σ (valuei − mean) = 0

这是 mean 作为 balance point 的定义性质。F25 Q4 直接考。

⚠️ 常见陷阱与考点

陷阱 1 · "95% chance interval contains parameter" 永远错
Parameter 固定、interval 一旦造好也固定 → 二者关系是确定的（要么包含，要么不包含），没有概率。
陷阱 2 · "95% of (population/sample) values lie in this interval" 永远错
CI 描述的是 parameter 的位置，不是 data values。Lec15 P14 直接展示了 CI 只覆盖 population histogram 极小一块。
陷阱 3 · 正确说法是「95% 的这种过程产生的 CI 会包含真值」
"Approximately 95% of intervals created through this process contain the true parameter." 这是唯一能 100% 算对的说法。
陷阱 4 · "95% of bootstrap medians lie in this interval" 对
这就是构造方式（中间 95%）。但题目里非常容易偷换成"sample medians" / "population medians" / "sample values"——这些都错！只有 "bootstrap medians" 或 "bootstrapped resample statistics" 才对。
陷阱 5 · 更高置信度 → 区间更宽（同一份数据）
99% CI > 95% CI > 80% CI。错说："99% CI 比 95% 窄" → False。
陷阱 6 · 更大样本 → 区间更窄
"n 从 100 变到 1000，CI 会变宽" → False。
陷阱 7 · skew 方向：mean 跟着 tail 跑
Right-skewed → mean > median。Left-skewed → mean < median。
陷阱 8 · SD 不能为负，可以为 0
SD = 0 当且仅当所有值相同。SD < 0 不可能（开根号结果非负）。
陷阱 9 · 加一个等于 mean 的值，会发生什么？
若 dataset 中加入一个值正好 = 当前 mean：mean 不变；新值的 deviation = 0；总平方和不变但 n+1 → variance 减小（除数变大、分子不变）。
陷阱 10 · Bootstrap 不能用来估 max/min
因为 bootstrap resample 的 max ≤ 原 sample 的 max（原 sample 决定了上限）。所以 boot 分布永远 underestimate population max。
💡 选 CI 解读题的速判法：
  1. 看到"95% chance"——错。
  2. 看到"95% of (data/values/people/salaries)"——错。
  3. 看到"95% of intervals"或"the process"——对。
  4. 看到"95% of bootstrap (medians/means/stats)"——对。

📝 往年真题

Spring 2024 Quiz 3 · Problem 2.3（CI 解读 · 经典 4 选项）

An 88% confidence interval for the mean meal price at Italian restaurants in this city is [25, 35]. Which of the following statements are valid interpretations of this confidence interval? Select all that apply.

  • (A) Approximately 88% of all Italian restaurants in this city have meal prices between $25 and $35.
  • (B) There is an 88% chance that the mean meal price at Italian restaurants in this city is between $25 and $35.
  • (C) If we repeated this procedure many times, about 88% of the resulting intervals would contain the true mean meal price.
  • (D) Approximately 88% of all bootstrapped sample means we computed fell between $25 and $35.
显示答案

答案：(C) 和 (D)

  • (A) ❌ CI 不是关于 data values 的，是关于 parameter 的。
  • (B) ❌ "88% chance the parameter is in [25, 35]"——错！parameter 固定，interval 固定。
  • (C) ✅ Confidence is in the process. 这是唯一一定对的说法。
  • (D) ✅ 这就是 88% CI 的构造方式：取 boot means 的中间 88%。
Fall 2025 Quiz 3 · Problem 1.3（CI 解读 · 比例版）

A 90% bootstrap CI for the population proportion of sleepy cats with ≥10 treats is [0.10, 0.15]. Which statements are valid? Select all that apply.

  • (A) There is a 90% chance the true proportion is between 0.10 and 0.15.
  • (B) About 90% of all cats have a treat count between 0.10 and 0.15.
  • (C) Roughly 90% of the bootstrap sample proportions fell in [0.10, 0.15].
  • (D) If we repeat the process (sample → bootstrap → CI) many times, ≈90% of these intervals would contain the true population proportion.
  • (E) The true population proportion is definitely in [0.10, 0.15].
显示答案

答案：(C) 和 (D)

(A) "X% chance"陷阱、(B) "X% of data values"陷阱、(E) "definitely"——都错。

  • (A) If we used a smaller original sample size, the CI would tend to be wider.
  • (B) About 95% of all such intervals (created through this process) will contain the true population mean.
  • (C) We could instead use the CLT to construct a CI based on the same single sample.
  • (D) The standard deviation of our sample is approximately the standard deviation of the population.
显示答案

注：本题为讲义改编版（从官方 6 选项中精简出 4 个"真命题"用于练习概念）。官方原题共 6 选项（含若干 false 干扰项，如"data is roughly normally distributed"），官方答案 = Option 4 + Option 5（即"could also use CLT"、"sample SD ≈ pop SD"——对应我们这里的 (C) 和 (D)）。完整原题见 practice_site/wi25-quiz3/index.html Problem 2.3。

Fall 2024 Quiz 3 · Problem 1.3（手算 variance）

Calculate the variance of the dataset [2, 4, 6, 8, 10] by hand. Show your work.

显示答案 
mean = (2+4+6+8+10)/5 = 6
deviations = [-4, -2, 0, 2, 4]
squared    = [16, 4, 0, 4, 16]
sum        = 40
variance   = 40/5 = 8

variance = 8（SD = √8 ≈ 2.83）

Fall 2024 Quiz 3 · Problem 1.4（变化分析）

Continuing from above: a sixth value equal to the mean (i.e. 6) is added to the dataset. How does the variance change?

  • (A) Increases　(B) Decreases　(C) Unchanged　(D) Cannot tell
显示答案

答案：(B) Decreases

新 mean = (2+4+6+8+10+6)/6 = 6（不变！）
新 deviations = [-4,-2,0,2,4, 0]
新 squared sum = 40（不变，因为新 deviation² = 0）
新 variance = 40 / 6 ≈ 6.67 < 8

加一个等于 mean 的值，分子（sum of squared deviations）不变但 n 变大 → variance 减小。

Fall 2025 Quiz 3 · Problem 4（多项 T/F · mean/median 性质）

For a dataset, the mean is 14.6 and the median is 16.3. Select all true statements.

  • (A) The distribution is left-skewed.
  • (B) There must exist a value between 14.6 and 16.3.
  • (C) The sum of (value − 14.6) over all values equals 0.
  • (D) Half the values are below 16.3.
  • (E) If we remove the largest value, the median will decrease.
显示答案

答案：(A), (C), (D)

  • (A) ✅ mean < median → 左偏（mean 被左尾拉走）。
  • (B) ❌ 不一定。比如 dataset 全是 14 和 17，mean = 15.5 不在数据里。具体到本题，并没有规定值要 cover 该区间。
  • (C) ✅ Sum of deviations from mean = 0 永远成立。
  • (D) ✅ Median 的定义。
  • (E) ❌ Median 对 outlier robust，移除最大值通常不影响（除非数据很少）。

答案：True

Winter 2024 Quiz 4 · Problem 2（SD 性质 T/F）

Select all TRUE statements about the standard deviation.

  • (A) The standard deviation can be 0.
  • (B) The standard deviation can be negative.
  • (C) If two datasets have the same SD, they must have the same mean.

显示答案

答案：(A) 和 (D)

  • (A) ✅ 当所有值相等，SD = 0。
